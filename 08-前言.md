# 第 0 章 前言

> **Abstract**: 算法犹如美妙的交响乐，每一行代码都像韵律般流淌。愿这本书在你的脑海中轻轻响起，留下独特而深刻的旋律。

## 0.1 关于本书

### 0.1.1 目标定位

本项目旨在创建一本**开源、免费、对新手友好**的数据结构与算法入门教程。

### 核心特色

| 特色 | 说明 |
|------|------|
| 动画图解 | 内容清晰易懂、学习曲线平滑，引导初学者探索知识地图 |
| 可运行代码 | 源代码一键运行，在练习中提升编程技能 |
| 互助学习 | 提倡读者互助，欢迎评论区提问与分享 |

### 读者对象

1. **算法初学者**：从未接触过算法，或已有刷题经验但认识模糊
2. **有一定基础者**：可作为"刷题工具库"或"算法字典"使用
3. **算法"大神"**：欢迎提出建议或参与创作

**前置条件**：至少具备任一语言的编程基础，能够阅读和编写简单代码。

### 0.1.2 内容结构

本书主要内容分为三大模块：

```
┌─────────────────────────────────────────────────────────────┐
│                     《Hello 算法》内容框架                   │
├─────────────────────────────────────────────────────────────┤
│  1. 复杂度分析                                                │
│     └── 时间复杂度、空间复杂度的推算方法、常见类型、示例            │

│     └── 各种结构的定义、优缺点、常用操作、典型应用、实现方法        │
│                                                              │

│     ├── 搜索、排序                                           │
```

## 0.2 如何使用本书

### 0.2.1 行文风格约定

| 标记类型 | 含义 |
|----------|------|
| 标题后标注 * | 选读章节，内容相对困难，可先跳过 |
| **黑体/下划线** | 专业术语，建议记住 |
| **加粗文字** | 重点内容和总结性语句 |
| "引号标注" | 有特指含义的词句 |

### 注释规范（Python版）

```python
""" 标题注释，用于标注函数、类、测试样例等 """

def function():
    # 内容注释，用于详解代码
    pass

""" 多行注释 """
```

### 0.2.2 学习方法

1. **以图为主、以文字为辅**：重点和难点知识主要通过动画图解展示
2. **动手实践**：
   - 参照代码自行敲一遍
   - 至少通读并运行所有代码
   - **与阅读代码相比，编写代码的过程往往能带来更多收获**

### 0.2.3 代码运行环境

**运行代码三步走**：

```bash
# 第一步：安装本地编程环境（参见附录教程）

# 第二步：克隆代码仓库
git clone https://github.com/krahets/hello-algo.git
# 或下载 ZIP 压缩包

# 第三步：运行源代码
# 在 codes/ 文件夹内找到对应的源代码文件，一键运行
```

**Python 可视化运行**：网页版支持基于 pythontutor 的可视化运行，可观察算法执行过程。

### 0.2.4 学习路线

从总体上，数据结构与算法的学习划分为三个阶段：

```
阶段一：算法入门
    │
    ├── 熟悉各种数据结构的特点和用法
    ├── 学习不同算法的原理、流程、用途和效率
    │
    ▼
阶段二：刷算法题  ← 本书主要覆盖这个阶段
    │
    ├── 从热门题目开刷，积累至少 100 道题目
    ├── 按"艾宾浩斯遗忘曲线"复习（3～5轮重复后牢记）
    │
    ▼
阶段三：搭建知识体系
    │
    ├── 阅读算法专栏、解题框架、算法教材
    ├── 进阶刷题策略：按专题分类、一题多解、一解多题
```

## 0.3 小结

### 重点回顾

1. 📖 本书主要受众是**算法初学者**，已有基础者可作为系统回顾工具
2. 📚 内容主要包括**复杂度分析、数据结构和算法**三部分
