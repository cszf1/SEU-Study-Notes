## 7.1 二叉树

**二叉树（binary tree）** 是一种非线性数据结构，代表"祖先"与"后代"之间的派生关系，体现了"一分为二"的分治逻辑。

### 7.1.1 二叉树常见术语

```python
class TreeNode:
    """ 二叉树节点类 """
    def __init__(self, val: int):
        self.val: int = val          # 节点值
        self.left: TreeNode | None = None   # 左子节点引用
        self.right: TreeNode | None = None  # 右子节点引用
```

**常用术语：**

| 术语 | 说明 |
|------|------|
| 根节点 (root) | 位于二叉树顶层的节点，没有父节点 |
| 叶节点 (leaf) | 没有子节点的节点 |
| 边 (edge) | 连接两个节点的线段 |
| 度 (degree) | 节点的子节点数量，二叉树中为 0、1、2 |
| 高度 (height) | 从根节点到最远叶节点所经过的边的数量 |
| 深度 (depth) | 从根节点到该节点所经过的边的数量 |

### 7.1.2 二叉树基本操作

**1. 初始化二叉树**

```python
# 初始化节点
n1 = TreeNode(val=1)
n2 = TreeNode(val=2)
n3 = TreeNode(val=3)
n4 = TreeNode(val=4)
n5 = TreeNode(val=5)

# 构建节点之间的引用（指针）
n1.left = n2
n1.right = n3
n2.left = n4
n2.right = n5
```

**2. 插入与删除节点**

```python
# 插入节点 P
p = TreeNode(0)
n1.left = p
p.left = n2

# 删除节点 P
n1.left = n2
```

### 7.1.3 常见二叉树类型

| 类型 | 说明 |
|------|------|
| **完美二叉树** | 所有层的节点都被完全填满，也称"满二叉树" |
| **完全二叉树** | 仅允许最底层节点不完全填满，且必须从左至右连续填充 |
| **完满二叉树** | 除叶节点外，所有节点都有两个子节点 |
| **平衡二叉树** | 任意节点的左子树和右子树高度之差的绝对值不超过 1 |

### 7.1.4 二叉树的退化

```
┌─────────────────────────────────────────────────────┐
│                    二叉树结构对比                    │
├─────────────────────────────────────────────────────┤
│                                                     │
│  完美二叉树（最佳）              链表（最差）       │
│       ○                          ○                  │
│      / \                        /                   │
│     ○   ○                      ○                    │
│    / \                        /                     │
│   ○   ○                      ○                      │
│                                                     │
│  • 完全平衡，高度 = O(log n)                        │
│  • 各项操作高效                                     │

│                          • 操作退化为线性           │
└─────────────────────────────────────────────────────┘
```

## 7.2 二叉树遍历

### 7.2.1 层序遍历（广度优先 BFS）

**思路：** 使用**队列**实现，"一圈一圈向外扩展"的逐层遍历方式。

```python
from collections import deque

def level_order(root: TreeNode | None) -> list[int]:
    """ 层序遍历 """
    if root is None:
        return []
    
    queue: deque[TreeNode] = deque()
    queue.append(root)
    res = []
    
    while queue:
        node: TreeNode = queue.popleft()  # 队列出队
        res.append(node.val)              # 保存节点值
        
        if node.left is not None:
            queue.append(node.left)       # 左子节点入队
        if node.right is not None:
            queue.append(node.right)      # 右子节点入队
    
    return res
```

**复杂度：**
- 时间：O(n)
- 空间：O(n)

### 7.2.2 前序、中序、后序遍历（深度优先 DFS）

**思路：** 递归实现，"先走到尽头，再回溯继续"。

```python
def pre_order(root: TreeNode | None):
    """ 前序遍历：根 -> 左 -> 右 """
    if root is None:
        return
    res.append(root.val)
    pre_order(root.left)
    pre_order(root.right)

def in_order(root: TreeNode | None):
    """ 中序遍历：左 -> 根 -> 右 """
    if root is None:
        return
    in_order(root.left)
    res.append(root.val)
    in_order(root.right)

def post_order(root: TreeNode | None):
    """ 后序遍历：左 -> 右 -> 根 """
    if root is None:
        return
    post_order(root.left)
    post_order(root.right)
    res.append(root.val)
```

**复杂度：**
- 时间：O(n)
- 空间：O(n)（最差退化为链表时）

## 7.3 二叉树数组表示

### 7.3.1 表示完美二叉树

给定完美二叉树，按**层序遍历**顺序存储在数组中：

```
数组索引关系：
• 节点 i 的左子节点索引 = 2i + 1
• 节点 i 的右子节点索引 = 2i + 2
• 节点 i 的父节点索引 = (i - 1) // 2
```

### 7.3.2 表示任意二叉树

对于非完美二叉树，使用 `None` 显式表示空位：

```python
# 二叉树的数组表示
tree = [1, 2, 3, 4, None, 6, 7, 8, 9, None, None, 12, None, None, 15]
```

### 7.3.3 优点与局限性

| 优点 | 局限性 |
|------|--------|
| 对缓存友好，访问遍历速度快 | 需要连续内存空间 |
| 不需要存储指针，节省空间 | 增删节点效率较低 |
| 允许随机访问节点 | 大量 None 时空间利用率低 |

## 7.4 二叉搜索树

**二叉搜索树 (BST)** 满足：
1. 左子树所有节点的值 < 根节点的值 < 右子树所有节点的值
2. 任意节点的左右子树也是二叉搜索树

### 7.4.1 二叉搜索树的操作

**1. 查找节点**

```python
def search(self, num: int) -> TreeNode | None:
    """ 查找节点 """
    cur = self._root
    while cur is not None:
        if cur.val < num:
            cur = cur.right
        elif cur.val > num:
            cur = cur.left
        else:
            break
    return cur
```

**2. 插入节点**

```python
def insert(self, num: int):
    """ 插入节点 """
    if self._root is None:
        self._root = TreeNode(num)
        return
    
    cur, pre = self._root, None
    while cur is not None:
        if cur.val == num:
            return  # 重复节点不插入
        pre = cur
        if cur.val < num:
            cur = cur.right
        else:
            cur = cur.left
    
    # 插入节点
    node = TreeNode(num)
    if pre.val < num:
        pre.right = node
    else:
        pre.left = node
```

**3. 删除节点**

根据待删除节点的子节点数量（0、1、2）分别处理：
- 度为 0：直接删除
- 度为 1：用子节点替换
- 度为 2：用中序遍历的下一个节点替换

### 7.4.2 二叉搜索树的效率

| 操作 | 无序数组 | 二叉搜索树 |
|------|---------|-----------|
| 查找 | O(n) | **O(log n)** |
| 插入 | O(1) | **O(log n)** |
| 删除 | O(n) | **O(log n)** |

⚠️ **退化问题**：频繁增删后可能退化为链表，退化为 O(n)

### 7.4.3 二叉搜索树常见应用

- 系统多级索引
- 搜索算法底层结构
- 数据流有序存储

## 7.5 AVL树（选读）

**AVL树** = 二叉搜索树 + 平衡二叉树，通过**旋转操作**保持平衡。

### 7.5.1 核心概念

**平衡因子** = 左子树高度 - 右子树高度，始终满足 -1 ≤ 平衡因子 ≤ 1

### 7.5.2 四种旋转操作

| 失衡情况 | 平衡因子 | 旋转方法 |
|---------|---------|---------|
| 左偏 | > 1 且子节点 ≥ 0 | 右旋 |
| 左偏 | > 1 且子节点 < 0 | 先左旋后右旋 |
| 右偏 | < -1 且子节点 ≤ 0 | 左旋 |
| 右偏 | < -1 且子节点 > 0 | 先右旋后左旋 |

### 7.5.3 AVL树典型应用

- 高频查找、低频增删场景
- 数据库索引系统
- 其他平衡二叉搜索树：红黑树（旋转更少，插入删除效率更高）

## 7.6 小结

### 重点回顾

1. ✅ **二叉树**是非线性数据结构，体现"一分为二"的分治逻辑
2. ✅ 二叉树遍历方式：
   - **层序遍历**（BFS）：队列实现，逐层推进
   - **前/中/后序遍历**（DFS）：递归实现，先走后回溯
3. ✅ **二叉搜索树 (BST)**：查找/插入/删除均为 O(log n)，退化后为 O(n)
4. ✅ **AVL树**：通过旋转保持平衡，确保 O(log n) 时间复杂度
5. ✅ 二叉树可以用**数组表示**，适合完全二叉树

### 二叉树 vs 链表效率对比

| 指标 | 完美二叉树 | 链表 |
|------|----------|------|
| 第 i 层节点数 | 2^(i-1) | 1 |
| 高度为 h 的节点总数 | 2^(h+1) - 1 | h + 1 |
| n 个节点的树高度 | log₂(n+1) - 1 | n - 1 |
