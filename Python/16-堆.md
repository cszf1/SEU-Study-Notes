## 8.1 堆的核心概念

### 8.1.1 什么是堆

**堆（heap）** 是一种满足特定条件的完全二叉树，主要可分为两种类型：

| 类型 | 定义 | 堆顶特点 |
|------|------|----------|
| **小顶堆（min heap）** | 任意节点的值 ≤ 其子节点的值 | 堆顶为最小值 |
| **大顶堆（max heap）** | 任意节点的值 ≥ 其子节点的值 | 堆顶为最大值 |

### 8.1.2 完全二叉堆的特性

```
完全二叉堆的示例结构：

        大顶堆                    小顶堆
           9                        1
         /   \                    /   \
       8      7                  3      5
      / \    / \                / \    / \
     6   5  4   3              6   7  9   8
```

**堆作为完全二叉树的特点：**

- 最底层节点靠左填充，其他层的节点都被填满
- 根节点称为"堆顶"，底层最靠右的节点称为"堆底"
- 对于大顶堆，堆顶元素的值是最大的；对于小顶堆，堆顶元素的值是最小的

### 8.1.3 堆与优先队列

> 堆通常用于实现**优先队列（priority queue）**，两者可视为等价的数据结构。

- 优先队列：具有优先级排序的队列
- 大顶堆 ≈ 元素按从大到小顺序出队的优先队列

## 8.2 堆的常用操作

堆的常用操作效率如下：

| 方法名 | 描述 | 时间复杂度 |
|--------|------|------------|
| `push()` | 元素入堆 | O(log n) |
| `pop()` | 堆顶元素出堆 | O(log n) |
| `peek()` | 访问堆顶元素 | O(1) |
| `size()` | 获取堆的元素数量 | O(1) |
| `isEmpty()` | 判断堆是否为空 | O(1) |

## 8.3 堆的存储与表示

由于堆是完全二叉树，**使用数组来存储堆**非常高效。

### 8.3.1 索引映射公式

给定索引 `i`，其关系如下：

```
父节点索引: parent(i) = (i - 1) // 2
左子节点:   left(i) = 2i + 1
右子节点:   right(i) = 2i + 2
```

### 8.3.2 数组表示示例

```
数组索引:    0    1    2    3    4    5    6
           ┌────┬────┬────┬────┬────┬────┬────┐
大顶堆:     │ 9  │ 8  │ 7  │ 6  │ 5  │ 4  │ 3  │
           └────┴────┴────┴────┴────┴────┴────┘
             │
           树结构:
               9(0)
             /    \
          8(1)    7(2)
          / \     / \
        6(3)5(4) 4(5)3(6)
```

## 8.4 元素入堆

### 8.4.1 入堆操作步骤

1. 将新元素添加到堆底（数组末尾）
2. 从底至顶执行**堆化（heapify）**
3. 比较插入节点与其父节点，如果更大则交换
4. 重复直到越过根节点或无需交换

### 8.4.2 时间复杂度

- 树的高度为 O(log n)
- 堆化最多进行 O(log n) 轮
- **入堆时间复杂度：O(log n)**

---

## 8.5 堆顶元素出堆

### 8.5.1 出堆操作步骤

1. 交换堆顶与堆底元素
2. 删除堆底（原来的堆顶）
3. 从顶至底执行堆化
4. 将根节点与子节点中较大的交换
5. 重复直到越过叶节点或无需交换

### 8.5.2 时间复杂度

- **出堆时间复杂度：O(log n)**

## 8.6 Python heapq 模块

Python 标准库 `heapq` 模块实现了堆队列算法。

### 8.6.1 核心函数

| 函数 | 功能 | 时间复杂度 |
|------|------|------------|
| `heapq.heapify(list)` | 将列表转换为堆 | O(n) |
| `heapq.heappush(heap, item)` | 插入元素 | O(log n) |
| `heapq.heappop(heap)` | 弹出最小元素 | O(log n) |
| `heapq.heappushpop(heap, item)` | 先推入再弹出 | O(log n) |
| `heapq.heapreplace(heap, item)` | 先弹出再推入 | O(log n) |
| `heapq.nlargest(n, iterable)` | 获取前 n 个最大元素 | O(n log k) |
| `heapq.nsmallest(n, iterable)` | 获取前 n 个最小元素 | O(n log k) |

### 8.6.2 基础使用

```python
import heapq

# 创建堆
heap = []
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappush(heap, 4)
print(heap)  # [1, 3, 4]  # 堆顶始终是最小值

# 从列表建堆（O(n)）
data = [3, 1, 4, 1, 5, 9, 2]
heapq.heapify(data)
print(data)  # [1, 1, 2, 3, 5, 9, 4]

# 弹出最小元素
smallest = heapq.heappop(data)
print(smallest)  # 1
print(data)      # [1, 2, 4, 3, 5, 9]
```

### 8.6.3 实现大顶堆

Python 的 `heapq` 默认实现小顶堆。通过对元素取负实现大顶堆：

```python
import heapq

# 方法1：存储负数
max_heap = []
heapq.heappush(max_heap, -5)
heapq.heappush(max_heap, -1)
heapq.heappush(max_heap, -3)
max_val = -heapq.heappop(max_heap)  # 5

# 方法2：flag 变量统一处理
max_heap, flag = [], -1
heapq.heappush(max_heap, flag * 5)
heapq.heappush(max_heap, flag * 1)
heapq.heappush(max_heap, flag * 3)
# 获取实际值
peek = flag * max_heap[0]  # 5
```

### 8.6.4 优先队列实现

```python
import heapq

# 使用元组 (优先级, 任务) 实现优先队列
tasks = []
heapq.heappush(tasks, (2, "写代码"))
heapq.heappush(tasks, (1, "修正Bug"))
heapq.heappush(tasks, (3, "写文档"))

while tasks:
    priority, task = heapq.heappop(tasks)
    print(f"执行 {task}，优先级 {priority}")
# 输出: 修正Bug(1) -> 写代码(2) -> 写文档(3)
```

### 8.6.5 处理优先级相同的任务

当优先级相同时，添加唯一计数器避免比较错误：

```python
import heapq
import itertools

counter = itertools.count()

def add_task(priority, task):
    """添加任务，优先级相同时按添加顺序处理"""
    entry = [priority, next(counter), task]
    heapq.heappush(tasks, entry)

tasks = []
add_task(5, "任务A")
add_task(3, "任务B")
add_task(5, "任务C")  # 与任务A优先级相同
```

### 8.6.6 获取 Top K 元素

```python
import heapq

numbers = [4, 1, 7, 3, 8, 5, 2, 6, 9, 0]

# 获取最大的3个元素
largest = heapq.nlargest(3, numbers)
print(largest)  # [9, 8, 7]

# 获取最小的3个元素
smallest = heapq.nsmallest(3, numbers)
print(smallest)  # [0, 1, 2]
```

## 8.7 堆的典型应用

### 8.7.1 优先队列

堆通常作为实现优先队列的首选数据结构：
- 入队和出队：O(log n)
- 建堆：O(n)
- 高效支持优先级动态调整

### 8.7.2 堆排序

```python
import heapq

def heap_sort(arr):
    """堆排序"""
    heap = arr.copy()
    heapq.heapify(heap)
    return [heapq.heappop(heap) for _ in range(len(heap))]

# 小顶堆得到升序，大顶堆（取负）得到降序
data = [5, 2, 8, 1, 9, 3]
sorted_asc = heap_sort(data)
print(sorted_asc)  # [1, 2, 3, 5, 8, 9]
```

### 8.7.3 获取 Top K 问题

**场景**：从大量数据中获取最大的 K 个元素

```python
import heapq

def top_k(items, k):
    """获取前 k 个最大的元素"""
    # 方法1：堆排序后取前k个 - O(n log n)
    return sorted(items, reverse=True)[:k]

def top_k_heap(items, k):
    """使用最小堆获取前 k 个最大的元素 - O(n log k)"""
    return heapq.nlargest(k, items)

# 测试
data = [1, 5, 2, 8, 3, 9, 1, 7]
print(top_k_heap(data, 3))  # [9, 8, 7]
```

## 8.8 小结

### 重点回顾

1. **堆的定义**：完全二叉树，分为小顶堆和大顶堆
2. **堆的存储**：数组实现，通过索引映射公式访问父子节点
3. **核心操作**：入堆 O(log n)、出堆 O(log n)、建堆 O(n)
4. **Python 实现**：使用 `heapq` 模块，默认小顶堆

### 算法对比

| 数据结构 | 获取最值 | 插入 | 删除 |
|----------|----------|------|------|
| 堆 | O(1) | O(log n) | O(log n) |
| 有序数组 | O(1) | O(n) | O(1) |
| 无序数组 | O(n) | O(1) | O(1) |

### 关键术语

| 术语 | 说明 |
|------|------|
| 堆 (heap) | 满足堆属性的完全二叉树 |
| 堆化 (heapify) | 修复堆属性的操作 |
| 堆顶 | 根节点，堆的最大/最小值所在 |
| 完全二叉堆 | 用数组实现的堆结构 |
