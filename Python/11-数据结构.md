## 3.1 数据结构分类

常见的数据结构包括**数组、链表、栈、队列、哈希表、树、堆、图**，它们可以从"逻辑结构"和"物理结构"两个维度进行分类。

### 3.1.1 逻辑结构：线性与非线性

逻辑结构揭示了数据元素之间的**逻辑关系**。

```
┌─────────────────────────────────────────────────────────────┐
│                    数据结构分类（逻辑结构）                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────┐         ┌─────────────────┐          │
│  │   线性结构       │         │   非线性结构     │          │
│  │  (一对一关系)    │         │                 │          │
│  └────────┬────────┘         └────────┬────────┘          │
│           │                          │                    │
│  ├── 数组  │                   ┌──────┴──────┐             │
│  ├── 链表  │                   │             │             │
│  ├── 栈    │                   │  树形结构    │  网状结构   │
│  ├── 队列  │                   │(一对多关系)  │(多对多关系) │
│  └── 哈希表│                   │  树、堆、哈希表│   图      │
│           │                   └──────────────┘             │
└─────────────────────────────────────────────────────────────┘
```

### 3.1.2 物理结构：连续与分散

物理结构描述了数据在**计算机内存中的存储方式**。

| 物理结构 | 存储方式 | 代表结构 | 特点 |
|---------|---------|---------|------|
| 连续空间存储 | 数组 | 元素紧密排列 | 随机访问快，插入删除慢 |
| 分散空间存储 | 链表 | 元素通过指针连接 | 插入删除快，随机访问慢 |

**关键结论**：所有数据结构都是基于**数组、链表**或两者的组合实现的。

```python
# 静态 vs 动态
数组 = "静态数据结构"  # 长度固定
链表 = "动态数据结构"  # 可动态调整长度
```

## 3.2 基本数据类型

基本数据类型是 CPU 可以直接进行运算的类型。

### 3.2.1 类型一览

| 类型 | 符号 | 典型占用空间 |
|------|------|-------------|
| 整数 | byte, short, **int**, long | 1/2/4/8 字节 |
| 浮点数 | **float**, double | 4/8 字节 |
| 字符 | char | 1/2 字节 |
| 布尔 | **bool** | 1 字节 |

### 3.2.2 Python 特性

```python
# Python 的特殊之处
numbers: list[int] = [0] * 5      # 整数可以是任意大小
decimals: list[float] = [0.0] * 5 # 浮点数是 64 位双精度
characters: list[str] = ['a'] * 5 # 没有 char，单字符是字符串
bools: list[bool] = [False] * 5   # 布尔类型

# Python 列表可混合存储不同类型
data = [0, 0.0, 'a', False, None]
```

### 3.2.3 基本数据类型 vs 数据结构

> **基本数据类型**提供数据的"**内容类型**"，**数据结构**提供数据的"**组织方式**"。

两者关系如同：
- 内容类型（int、float）= 货物种类
- 数据结构（数组、链表）= 货架排列方式

## 3.3 数字编码*（选读）

### 3.3.1 原码、反码和补码

计算机中整数以**补码**形式存储。

```
原码 → 反码 → 补码
  │       │
  │       └── 符号位不变，其余位取反
  └── 最高位为符号位
```

**为什么使用补码？**

1. **统一加减法**：可以用同一套电路处理加法和减法
2. **消除正负零歧义**：0 只有一种表示
3. **简化硬件设计**：只需实现加法器

```python
# Python 中的补码演示（了解即可）
# 正数：补码 = 原码
# 负数：补码 = 反码 + 1

# 示例：-1 的补码计算
# 原码: 10000001
# 反码: 11111110  
# 补码: 11111111
```

### 3.3.2 浮点数编码（IEEE 754）

float 的结构（32位）：

| 区域 | 长度 | 作用 |
|------|------|------|
| 符号位 S | 1位 | 表示正负 |
| 指数位 E | 8位 | 表示数量级 |
| 分数位 N | 23位 | 表示精度 |

**计算公式**：`val = (-1)^S × 2^(E-127) × (1.N)`

```python
# 为什么 float 取值范围比 int 大？
# 因为指数位使得可以表示非常大的数
# 代价：牺牲了精度（大数之间间距变大）

# 示例
import struct
print(struct.pack('>f', 1.0))   # float32
print(struct.pack('>d', 1.0))   # float64(double)
```

## 3.4 字符编码*（选读）

### 3.4.1 编码发展历程

```
ASCII (7位/128字符)
    ↓
EASCII (8位/256字符)
    ↓
GBK (支持中文)
    ↓
Unicode (统一编码)
    ↓
UTF-8 (变长编码，1-4字节)
```

### 3.4.2 UTF-8 编码规则

| 字节数 | 编码格式 | 适用字符 |
|--------|----------|----------|
| 1字节 | `0xxxxxxx` | ASCII |
| 2字节 | `110xxxxx 10xxxxxx` | 拉丁字母 |
| 3字节 | `1110xxxx 10xxxxxx 10xxxxxx` | 中文 |
| 4字节 | `11110xxx 10xxxxxx 10xxxxxx 10xxxxxx` | 生僻字 |

```python
# Python 中的字符串编码
s = "Hello 算法"

# UTF-8 编码（国际标准）
utf8_bytes = s.encode('utf-8')
print(f"UTF-8: {len(utf8_bytes)} 字节")

# 查看字符的 Unicode 码点
for char in s:
    print(f"'{char}': U+{ord(char):04X}")
```

### 3.4.3 各语言的字符编码

| 语言 | 编码方式 | 特点 |
|------|----------|------|
| Java/C#/JS | UTF-16 | 等长编码，2/4字节 |
| Python | Unicode（灵活） | 1-4字节自适应 |
| Go/Rust | UTF-8 | 内部存储 |
| C/C++ | 取决于平台 | 1字节 |

## 3.5 小结

### 重点回顾

1. **数据结构分类**
   - 逻辑结构：线性（数组、链表）vs 非线性（树、图）
   - 物理结构：连续（数组）vs 分散（链表）

2. **基本数据类型**
   - 提供数据的"内容类型"
   - Python 中 int 可以是任意大小

3. **数字编码**
   - 计算机使用补码存储整数
   - 浮点数采用 IEEE 754 标准

4. **字符编码**
   - UTF-8 是最广泛使用的 Unicode 编码
   - 可变长度，ASCII 兼容

### 核心概念图

```
┌─────────────────────────────────────────────────────────────┐
│                        计算机数据                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  基本数据类型 ←────────── 内容类型                          │
│  (int, float, bool)                                         │
│       │                                                     │
│       ▼                                                     │
│  数据结构 ←──────────── 组织方式                            │
│  (数组, 链表, 树, 图)                                        │
│       │                                                     │
│       ▼                                                     │
│  存储方式 ←──────────── 物理结构                           │
│  (连续/分散空间)                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
