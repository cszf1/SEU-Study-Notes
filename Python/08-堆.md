# 第8章 堆（Heap）

> 堆就像是山岳峰峦，层叠起伏、形态各异。座座山峰高低错落，而最高的山峰总是最先映入眼帘。

## 8.1 堆

### 定义

堆（Heap）是一种满足特定条件的**完全二叉树**，主要可分为两种类型：

| 类型 | 定义 |
|------|------|
| 小顶堆（min heap） | 任意节点的值 ≤ 其子节点的值 |
| 大顶堆（max heap） | 任意节点的值 ≥ 其子节点的值 |

### 堆的特性

- **完全二叉树**：最底层节点靠左填充，其他层的节点都被填满
- **堆顶**：二叉树的根节点，值最大（对于大顶堆）或最小（对于小顶堆）
- **堆底**：底层最靠右的节点

---

## 8.1.1 堆的常用操作

堆通常用于实现**优先队列**。堆的常用操作及其时间复杂度如下：

| 方法名 | 描述 | 时间复杂度 |
|--------|------|-----------|
| `push()` | 元素入堆 | O(log n) |
| `pop()` | 堆顶元素出堆 | O(log n) |
| `peek()` | 访问堆顶元素 | O(1) |
| `size()` | 获取堆的元素数量 | O(1) |
| `isEmpty()` | 判断堆是否为空 | O(1) |

### Python 中的堆

Python 的 `heapq` 模块默认实现**小顶堆**。若要实现大顶堆，可以将元素取负后再入堆。

```python
import heapq

# 初始化小顶堆
min_heap = []

# 初始化大顶堆（通过取负实现）
max_heap = []
flag = -1  # 取负 flag = -1

# 元素入堆
heapq.heappush(max_heap, flag * 1)   # 入堆 -1
heapq.heappush(max_heap, flag * 3)   # 入堆 -3
heapq.heappush(max_heap, flag * 2)   # 入堆 -2

# 获取堆顶元素
peek = -max_heap[0]  # 3

# 堆顶元素出堆（形成从大到小的序列）
val = -heapq.heappop(max_heap)  # 3
val = -heapq.heappop(max_heap)  # 2
val = -heapq.heappop(max_heap)  # 1

# 获取堆大小
size = len(max_heap)

# 判断堆是否为空
is_empty = not max_heap

# 输入列表并建堆
min_heap = [1, 3, 2, 5, 4]
heapq.heapify(min_heap)
```

## 8.1.2 堆的实现

### 1. 堆的存储与表示

完全二叉树非常适合用**数组**来表示。给定索引 `i`，其索引映射公式为：

| 关系 | 索引公式 |
|------|----------|
| 左子节点 | `2 * i + 1` |
| 右子节点 | `2 * i + 2` |
| 父节点 | `(i - 1) // 2` |

```python
class MaxHeap:
    """大顶堆实现"""
    
    def __init__(self, nums: list[int] = None):
        """构造方法，根据输入列表建堆"""
        if nums:
            self.max_heap = nums
            # 堆化除叶节点以外的所有节点
            for i in range(self.parent(len(nums) - 1), -1, -1):
                self.sift_down(i)
        else:
            self.max_heap = []
    
    def size(self) -> int:
        """获取堆的元素数量"""
        return len(self.max_heap)
    
    def is_empty(self) -> bool:
        """判断堆是否为空"""
        return len(self.max_heap) == 0
    
    def parent(self, i: int) -> int:
        """获取父节点索引"""
        return (i - 1) // 2
    
    def left(self, i: int) -> int:
        """获取左子节点索引"""
        return 2 * i + 1
    
    def right(self, i: int) -> int:
        """获取右子节点索引"""
        return 2 * i + 2
    
    def swap(self, i: int, j: int):
        """交换两节点"""
        self.max_heap[i], self.max_heap[j] = \
            self.max_heap[j], self.max_heap[i]
```

### 2. 访问堆顶元素

堆顶元素即为二叉树的根节点，也就是列表的首个元素：

```python
def peek(self) -> int:
    """访问堆顶元素"""
    return self.max_heap[0]
```

### 3. 元素入堆

入堆操作步骤：
1. 将元素添加到堆底
2. 从底至顶执行**堆化**（sift_up）
3. 比较插入节点与其父节点的值，如果插入节点更大，则交换
4. 循环执行直至越过根节点或遇到无须交换的节点

```python
def push(self, val: int):
    """元素入堆"""
    # 添加节点
    self.max_heap.append(val)
    # 从底至顶堆化
    self.sift_up(self.size() - 1)

def sift_up(self, i: int):
    """从节点 i 开始，从底至顶堆化"""
    while True:
        p = self.parent(i)  # 获取父节点
        # 当"越过根节点"或"节点无须修复"时，结束堆化
        if p < 0 or self.max_heap[i] <= self.max_heap[p]:
            break
        # 交换两节点
        self.swap(i, p)
        # 循环向上堆化
        i = p
```

**时间复杂度**：O(log n)

### 4. 堆顶元素出堆

出堆操作步骤：
1. 交换堆顶元素与堆底元素
2. 删除堆底（原来的堆顶元素）
3. 从根节点开始，从顶至底执行**堆化**（sift_down）

```python
def pop(self) -> int:
    """元素出堆"""
    if self.is_empty():
        raise IndexError("堆为空")
    
    # 交换根节点与最右叶节点
    self.swap(0, self.size() - 1)
    # 删除节点
    val = self.max_heap.pop()
    # 从顶至底堆化
    self.sift_down(0)
    # 返回堆顶元素
    return val

def sift_down(self, i: int):
    """从节点 i 开始，从顶至底堆化"""
    while True:
        l = self.left(i)
        r = self.right(i)
        ma = i
        
        # 判断节点 i, l, r 中值最大的节点
        if l < self.size() and self.max_heap[l] > self.max_heap[ma]:
            ma = l
        if r < self.size() and self.max_heap[r] > self.max_heap[ma]:
            ma = r
        
        # 若节点 i 最大或索引 l, r 越界，则无须继续堆化
        if ma == i:
            break
        
        # 交换两节点
        self.swap(i, ma)
        # 循环向下堆化
        i = ma
```

**时间复杂度**：O(log n)

## 8.2 建堆操作

### 方法一：借助入堆操作

遍历列表，依次对每个元素执行入堆操作。

```python
def build_heap(nums: list[int]) -> list[int]:
    """方法一：入堆建堆"""
    heap = []
    for num in nums:
        heapq.heappush(heap, num)
    return heap
```

**时间复杂度**：O(n log n)

### 方法二：遍历堆化

1. 将列表所有元素原封不动地添加到堆中
2. 倒序遍历堆，依次对每个非叶节点执行"从顶至底堆化"

```python
def build_heap_optimized(nums: list[int]) -> list[int]:
    """方法二：遍历堆化建堆"""
    # 最后一个非叶节点的索引
    n = len(nums)
    start = (n - 1 - 1) // 2  # parent(n - 1)
    
    # 倒序堆化
    for i in range(start, -1, -1):
        # sift_down 操作
        while True:
            l = 2 * i + 1
            r = 2 * i + 2
            ma = i
            if l < n and nums[l] > nums[ma]:
                ma = l
            if r < n and nums[r] > nums[ma]:
                ma = r
            if ma == i:
                break
            nums[i], nums[ma] = nums[ma], nums[i]
            i = ma
    
    return nums
```

**时间复杂度**：O(n)

### 复杂度分析

倒序堆化建堆的时间复杂度为 O(n)，因为：
- 底层节点数量多但堆化高度小
- 顶层节点数量少但堆化高度大
- 通过数学推导可得总体复杂度为 O(n)

## 8.3 Top-k 问题

**问题**：给定一个长度为 n 的无序数组 nums，请返回数组中最大的 k 个元素。

### 方法比较

| 方法 | 时间复杂度 | 适用场景 |
|------|-----------|----------|
| 遍历选择 | O(nk) | k << n |
| 先排序 | O(n log n) | 需要完整排序 |
| **堆** | **O(n log k)** | **通用场景** |

### 堆解法

```python
import heapq

def top_k_heap(nums: list[int], k: int) -> list[int]:
    """基于堆查找数组中最大的 k 个元素"""
    
    # 初始化小顶堆
    heap = []
    
    # 将数组的前 k 个元素入堆
    for i in range(k):
        heapq.heappush(heap, nums[i])
    
    # 从第 k+1 个元素开始，保持堆的长度为 k
    for i in range(k, len(nums)):
        # 若当前元素大于堆顶元素，则将堆顶元素出堆、当前元素入堆
        if nums[i] > heap[0]:
            heapq.heappop(heap)
            heapq.heappush(heap, nums[i])
    
    return heap
```

**思路**：
1. 维护一个大小为 k 的小顶堆
2. 遍历剩余元素，若比堆顶大则替换
3. 最终堆中保存最大的 k 个元素

**时间复杂度**：O(n log k)

## 8.4 堆的常见应用

1. **优先队列**：堆通常作为实现优先队列的首选数据结构
2. **堆排序**：基于堆的排序算法，时间复杂度 O(n log n)
3. **获取 Top-k 元素**：经典算法问题

### 补充：堆与内存堆的区别

> Q：数据结构的"堆"与内存管理的"堆"是同一个概念吗？

**不是同一个概念**，只是碰巧都叫"堆"。

- **数据结构堆**：完全二叉树，用于实现优先队列
- **内存堆**：动态内存分配区域，用于存储程序运行时的数据

## 本章小结

1. **堆的类型**：大顶堆和小顶堆，堆顶元素分别是最大/最小的
2. **时间复杂度**：入堆 O(log n)、出堆 O(log n)、访问堆顶 O(1)
3. **存储方式**：完全二叉树非常适合用数组表示
4. **堆化操作**：从底至顶堆化（入堆）和从顶至底堆化（出堆）
5. **建堆复杂度**：O(n)
6. **Top-k 问题**：使用小顶堆，时间复杂度 O(n log k)
