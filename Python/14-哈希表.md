## 6.1 哈希表

### 基本概念

哈希表（Hash Table），又称散列表，通过建立键（Key）与值（Value）之间的映射，实现高效的元素查询。

**核心优势**：输入一个键 Key，可以在 **O(1)** 时间内获取对应的值 Value。

### 效率对比

| 操作 | 数组 | 链表 | 哈希表 |
|------|------|------|--------|
| 查找元素 | O(n) | O(n) | **O(1)** |
| 添加元素 | O(1) | O(1) | **O(1)** |
| 删除元素 | O(n) | O(n) | **O(1)** |

### 6.1.1 常用操作

```python
# 初始化哈希表
hmap: dict = {}

# 添加操作
hmap[12836] = "小哈"
hmap[15937] = "小啰"
hmap[16750] = "小算"
hmap[13276] = "小法"
hmap[10583] = "小鸭"

# 查询操作
name: str = hmap[15937]  # 返回 "小啰"

# 删除操作
hmap.pop(10583)

# 遍历哈希表
for key, value in hmap.items():
    print(key, "->", value)

for key in hmap.keys():
    print(key)

for value in hmap.values():
    print(value)
```

### 6.1.2 简单实现

哈希函数的作用是将一个较大的输入空间映射到较小的输出空间（桶数组索引）。

**核心公式**：
```
index = hash(key) % capacity
```

```python
class Pair:
    """键值对"""
    def __init__(self, key: int, val: str):
        self.key = key
        self.val = val

class ArrayHashMap:
    """基于数组实现的哈希表"""
    def __init__(self):
        # 初始化数组，包含100个桶
        self.buckets: list[Pair | None] = [None] * 100
    
    def hash_func(self, key: int) -> int:
        """哈希函数"""
        index = key % 100
        return index
    
    def get(self, key: int) -> str | None:
        """查询操作"""
        index: int = self.hash_func(key)
        pair: Pair = self.buckets[index]
        if pair is None:
            return None
        return pair.val
    
    def put(self, key: int, val: str):
        """添加和更新操作"""
        pair = Pair(key, val)
        index: int = self.hash_func(key)
        self.buckets[index] = pair
    
    def remove(self, key: int):
        """删除操作"""
        index: int = self.hash_func(key)
        self.buckets[index] = None
    
    def entry_set(self) -> list[Pair]:
        """获取所有键值对"""
        result: list[Pair] = []
        for pair in self.buckets:
            if pair is not None:
                result.append(pair)
        return result
    
    def key_set(self) -> list[int]:
        """获取所有键"""
        result = []
        for pair in self.buckets:
            if pair is not None:
                result.append(pair.key)
        return result
    
    def value_set(self) -> list[str]:
        """获取所有值"""
        result = []
        for pair in self.buckets:
            if pair is not None:
                result.append(pair.val)
        return result
```

### 6.1.3 哈希冲突与扩容

**哈希冲突（Hash Collision）**：多个输入对应同一输出的情况。

例如：`12836 % 100 = 36`，`20336 % 100 = 36`，两者指向同一桶。

**扩容机制**：
- 哈希表容量越大，冲突概率越低
- 负载因子（Load Factor）= 元素数量 / 桶数量
- 当负载因子超过阈值（Java 中为 0.75）时触发扩容

## 6.2 哈希冲突

### 6.2.1 链式地址

将单个元素转换为链表，所有冲突键值对存储在同一链表中。

```python
class HashMapChaining:
    """链式地址哈希表"""
    def __init__(self):
        self.size = 0          # 键值对数量
        self.capacity = 4      # 哈希表容量
        self.load_thres = 2.0 / 3.0  # 触发扩容的负载因子阈值
        self.extend_ratio = 2  # 扩容倍数
        self.buckets = [[] for _ in range(self.capacity)]  # 桶数组
    
    def hash_func(self, key: int) -> int:
        return key % self.capacity
    
    def load_factor(self) -> float:
        return self.size / self.capacity
    
    def get(self, key: int) -> str | None:
        index = self.hash_func(key)
        bucket = self.buckets[index]
        for pair in bucket:
            if pair.key == key:
                return pair.val
        return None
    
    def put(self, key: int, val: str):
        if self.load_factor() > self.load_thres:
            self.extend()
        index = self.hash_func(key)
        bucket = self.buckets[index]
        for pair in bucket:
            if pair.key == key:
                pair.val = val
                return
        pair = Pair(key, val)
        bucket.append(pair)
        self.size += 1
    
    def remove(self, key: int):
        index = self.hash_func(key)
        bucket = self.buckets[index]
        for pair in bucket:
            if pair.key == key:
                bucket.remove(pair)
                self.size -= 1
                break
    
    def extend(self):
        buckets = self.buckets
        self.capacity *= self.extend_ratio
        self.buckets = [[] for _ in range(self.capacity)]
        self.size = 0
        for bucket in buckets:
            for pair in bucket:
                self.put(pair.key, pair.val)
```

**特点**：
- 链表过长时查询效率退化至 O(n)
- 可将链表转换为 AVL 树或红黑树，优化至 O(log n)

### 6.2.2 开放寻址

通过多次探测来处理哈希冲突，不引入额外数据结构。

#### 1. 线性探测

- 插入/查找时，若桶被占用，向后线性遍历（步长为1）
- 问题：容易产生"聚集现象"

```python
class HashMapOpenAddressing:
    """开放寻址哈希表"""
    def __init__(self):
        self.size = 0
        self.capacity = 4
        self.load_thres = 2.0 / 3.0
        self.extend_ratio = 2
        self.buckets: list[Pair | None] = [None] * self.capacity
        self.TOMBSTONE = Pair(-1, "-1")  # 删除标记
    
    def hash_func(self, key: int) -> int:
        return key % self.capacity
    
    def load_factor(self) -> float:
        return self.size / self.capacity
    
    def find_bucket(self, key: int) -> int:
        """搜索key对应的桶索引"""
        index = self.hash_func(key)
        first_tombstone = -1
        while self.buckets[index] is not None:
            if self.buckets[index].key == key:
                if first_tombstone != -1:
                    self.buckets[first_tombstone] = self.buckets[index]
                    self.buckets[index] = self.TOMBSTONE
                    return first_tombstone
                return index
            if first_tombstone == -1 and self.buckets[index] is self.TOMBSTONE:
                first_tombstone = index
            index = (index + 1) % self.capacity
        return index if first_tombstone == -1 else first_tombstone
    
    def get(self, key: int) -> str | None:
        index = self.find_bucket(key)
        if self.buckets[index] not in [None, self.TOMBSTONE]:
            return self.buckets[index].val
        return None
    
    def put(self, key: int, val: str):
        if self.load_factor() > self.load_thres:
            self.extend()
        index = self.find_bucket(key)
        if self.buckets[index] not in [None, self.TOMBSTONE]:
            self.buckets[index].val = val
            return
        self.buckets[index] = Pair(key, val)
        self.size += 1
    
    def remove(self, key: int):
        index = self.find_bucket(key)
        if self.buckets[index] not in [None, self.TOMBSTONE]:
            self.buckets[index] = self.TOMBSTONE
            self.size -= 1
    
    def extend(self):
        buckets_tmp = self.buckets
        self.capacity *= self.extend_ratio
        self.buckets = [None] * self.capacity
        self.size = 0
        for pair in buckets_tmp:
            if pair not in [None, self.TOMBSTONE]:
                self.put(pair.key, pair.val)
```

#### 2. 平方探测

探测步长为探测次数的平方：1, 4, 9, ...

#### 3. 多次哈希

使用多个哈希函数 f₁(x), f₂(x), f₃(x)... 进行探测。

### 6.2.3 编程语言的选择

| 语言 | 实现策略 |
|------|----------|
| Python | 开放寻址（伪随机数探测） |
| Java | 链式地址（链表长度≥8时转为红黑树） |
| Go | 链式地址（每个桶最多8个键值对） |

## 6.3 哈希算法

### 6.3.1 哈希算法的目标

- **确定性**：相同输入产生相同输出
- **效率高**：计算开销小
- **均匀分布**：键值对均匀分布，冲突概率低

### 6.3.2 简单哈希算法

```python
def add_hash(key: str) -> int:
    """加法哈希"""
    hash = 0
    modulus = 1000000007
    for c in key:
        hash += ord(c)
    return hash % modulus

def mul_hash(key: str) -> int:
    """乘法哈希"""
    hash = 0
    modulus = 1000000007
    for c in key:
        hash = 31 * hash + ord(c)
    return hash % modulus

def xor_hash(key: str) -> int:
    """异或哈希"""
    hash = 0
    modulus = 1000000007
    for c in key:
        hash ^= ord(c)
    return hash % modulus

def rot_hash(key: str) -> int:
    """旋转哈希"""
    hash = 0
    modulus = 1000000007
    for c in key:
        hash = (hash << 4) ^ (hash >> 28) ^ ord(c)
    return hash % modulus
```

**为什么用大质数取模？**
- 质数不与其他数字存在公约数
- 减少周期性模式，避免哈希冲突聚集

### 6.3.3 常见哈希算法

| 算法 | 推出时间 | 输出长度 | 安全等级 | 应用 |
|------|----------|----------|----------|------|
| MD5 | 1992 | 128 bit | 低（已弃用） | 文件完整性检查 |
| SHA-1 | 1995 | 160 bit | 低（已弃用） | - |
| SHA-256 | 2002 | 256 bit | 高 | 加密货币、数字签名 |
| SHA-3 | 2008 | 224/256/384/512 bit | 高 | 替代SHA-2 |

### 6.3.4 Python内置哈希

```python
num = 3
hash_num = hash(num)        # 整数: 3

bol = True
hash_bol = hash(bol)        # 布尔: 1

dec = 3.14159
hash_dec = hash(dec)        # 小数: 326484311674566659

str = "Hello 算法"
hash_str = hash(str)       # 字符串: 4617003410720528961

tup = (12836, "小哈")
hash_tup = hash(tup)       # 元组: 1029005403108185979

obj = ListNode(0)
hash_obj = hash(obj)       # 对象: 基于内存地址
```

**注意**：
- 只有不可变对象可作为哈希表的 key
- Python 每次启动会为字符串哈希函数加入随机盐值，防 HashDoS 攻击

## 6.4 小结

### 重点回顾

1. **核心特性**：哈希表查询/添加/删除均为 **O(1)** 时间复杂度
2. **哈希函数**：`index = hash(key) % capacity`，将 key 映射到桶索引
3. **哈希冲突**：多个 key 映射到同一索引
4. **负载因子**：元素数量/桶数量，用于触发扩容
5. **链式地址**：链表存储冲突元素，可优化为红黑树
6. **开放寻址**：线性探测、平方探测、多次哈希
7. **哈希算法**：加法哈希、乘法哈希、异或哈希、旋转哈希
8. **常见算法**：MD5、SHA-1、SHA-2、SHA-3

### Q&A

**Q: 哈希表何时退化为 O(n)？**
A: 哈希冲突严重时。设计良好的哈希函数和合理的容量设置可保持 O(1)。

**Q: 为什么不用 f(x) = x？**
A: 输入空间远大于输出空间，哈希表目标是将大状态空间映射到小空间，同时提供 O(1) 查询。

**Q: 为什么扩容能缓解冲突？**
A: 取模的模数改变后，原先落在同一桶的 key 可能被分配到多个桶中。
