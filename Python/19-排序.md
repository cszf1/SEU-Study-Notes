## 11.1 排序算法

### 排序算法的评价维度

| 维度 | 说明 |
|------|------|
| **运行效率** | 时间复杂度，排序速度 |
| **就地性** | 空间复杂度，是否使用额外空间 |
| **稳定性** | 相等元素的相对顺序是否保持 |
| **自适应性** | 时间复杂度是否受输入数据影响 |
| **通用性** | 是否能处理任意数据类型 |

### 排序算法分类

```
排序算法
├── 比较类排序（通过比较决定元素相对次序）
│   ├── 交换排序：冒泡排序、快速排序
│   ├── 插入排序：插入排序、希尔排序
│   ├── 选择排序：选择排序、堆排序
│   └── 归并排序：归并排序
└── 非比较排序（不通过比较）
    ├── 桶排序
    ├── 计数排序
    └── 基数排序
```

## 11.2 选择排序

### 算法思想

开启一个循环，每轮从未排序区间选择最小的元素，将其放到已排序区间的末尾。

### 算法流程

1. 初始状态下，所有元素未排序，未排序区间为 `[0, n-1]`
2. 选取区间 `[0, n-1]` 中的最小元素，与索引 0 交换
3. 选取区间 `[1, n-1]` 中的最小元素，与索引 1 交换
4. 以此类推，经过 `n-1` 轮后数组排序完成

### Python 代码实现

```python
def selection_sort(nums: list[int]):
    """选择排序"""
    n = len(nums)
    # 外循环：未排序区间为 [i, n-1]
    for i in range(n - 1):
        # 内循环：找到未排序区间内的最小元素
        k = i
        for j in range(i + 1, n):
            if nums[j] < nums[k]:
                k = j  # 记录最小元素的索引
        # 将该最小元素与未排序区间的首个元素交换
        nums[i], nums[k] = nums[k], nums[i]


# 测试
if __name__ == "__main__":
    nums = [4, 1, 3, 1, 5, 2]
    selection_sort(nums)
    print(f"选择排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n²)，非自适应排序 |
| 空间复杂度 | O(1)，原地排序 |
| 稳定性 | 非稳定排序 |

## 11.3 冒泡排序

### 算法思想

通过连续地比较与交换相邻元素实现排序。这个过程就像气泡从底部升到顶部一样。

### 算法流程

1. 对 n 个元素执行"冒泡"，将最大元素交换至最右端
2. 对剩余 n-1 个元素执行"冒泡"，将第二大元素交换至正确位置
3. 以此类推，经过 n-1 轮后排序完成

### Python 代码实现

```python
def bubble_sort(nums: list[int]):
    """冒泡排序"""
    n = len(nums)
    # 外循环：未排序区间为 [0, i]
    for i in range(n - 1, 0, -1):
        # 内循环：将未排序区间 [0, i] 中的最大元素交换至最右端
        for j in range(i):
            if nums[j] > nums[j + 1]:
                nums[j], nums[j + 1] = nums[j + 1], nums[j]


# 测试
if __name__ == "__main__":
    nums = [3, 1, 2, 5, 4]
    bubble_sort(nums)
    print(f"冒泡排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n²)，最佳 O(n)（已排序时） |
| 空间复杂度 | O(1)，原地排序 |
| 稳定性 | 稳定排序 |
| 自适应性 | 是（已排序数组只需 O(n)） |

## 11.4 插入排序

### 算法思想

在未排序区间选择一个基准元素，将该元素与其左侧已排序区间的元素逐一比较大小，并将该元素插入到正确的位置。

### 算法流程

1. 初始状态下，第一个元素为已排序区间
2. 选取未排序区间的第一个元素作为 base
3. 在已排序区间从后向前比较，找到小于等于 base 的元素位置
4. 将该位置之后的元素右移，插入 base

### Python 代码实现

```python
def insertion_sort(nums: list[int]):
    """插入排序"""
    n = len(nums)
    # 外循环： base 为待插入元素
    for i in range(1, n):
        base = nums[i]
        j = i - 1
        # 内循环：寻找插入位置，同时后移元素
        while j >= 0 and nums[j] > base:
            nums[j + 1] = nums[j]
            j -= 1
        nums[j + 1] = base


# 测试
if __name__ == "__main__":
    nums = [4, 1, 3, 1, 5, 2]
    insertion_sort(nums)
    print(f"插入排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n²)，最佳 O(n)（已排序时） |
| 空间复杂度 | O(1)，原地排序 |
| 稳定性 | 稳定排序 |
| 自适应性 | 是 |

## 11.5 快速排序

### 算法思想

选择数组中的某个元素作为"基准数"，将所有小于基准数的元素移到其左侧，大于基准数的元素移到其右侧。

### 哨兵划分

```python
def partition(nums: list[int], left: int, right: int) -> int:
    """哨兵划分"""
    # 以 nums[left] 为基准数
    i, j = left, right
    while i < j:
        while i < j and nums[j] >= nums[left]:
            j -= 1  # 从右向左找首个小于基准数的元素
        while i < j and nums[i] <= nums[left]:
            i += 1  # 从左向右找首个大于基准数的元素
        nums[i], nums[j] = nums[j], nums[i]  # 交换这两个元素
    nums[i], nums[left] = nums[left], nums[i]  # 将基准数交换至分界线
    return i  # 返回基准数的索引
```

### 快速排序（递归实现）

```python
def quick_sort(nums: list[int], left: int, right: int):
    """快速排序"""
    # 终止条件
    if left >= right:
        return
    # 哨兵划分
    pivot = partition(nums, left, right)
    # 递归排序左子数组
    quick_sort(nums, left, pivot - 1)
    # 递归排序右子数组
    quick_sort(nums, pivot + 1, right)


# 测试
if __name__ == "__main__":
    nums = [2, 4, 1, 0, 3, 5]
    quick_sort(nums, 0, len(nums) - 1)
    print(f"快速排序完成后 nums = {nums}")
```

### 快速排序（尾递归优化）

```python
def quick_sort_tail_call(nums: list[int], left: int, right: int):
    """快速排序（尾递归优化）"""
    while left < right:
        # 哨兵划分
        pivot = partition(nums, left, right)
        # 对较短的子数组递归，减少栈深度
        if pivot - left < right - pivot:
            quick_sort_tail_call(nums, left, pivot - 1)
            left = pivot + 1
        else:
            quick_sort_tail_call(nums, pivot + 1, right)
            right = pivot - 1
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | 平均 O(n log n)，最差 O(n²) |
| 空间复杂度 | O(log n)（递归深度） |
| 稳定性 | 非稳定排序 |
| 自适应性 | 是 |

## 11.6 归并排序

### 算法思想

基于分治策略，包含"划分"和"合并"两个阶段：

1. **划分阶段**：递归地将数组从中点分开
2. **合并阶段**：将两个有序数组合并为一个有序数组

### Python 代码实现

```python
def merge(nums: list[int], left: int, mid: int, right: int):
    """合并左子数组和右子数组"""
    # 左子数组区间为 [left, mid], 右子数组区间为 [mid+1, right]
    tmp = [0] * (right - left + 1)
    i, j, k = left, mid + 1, 0
    
    # 合并两个有序数组
    while i <= mid and j <= right:
        if nums[i] <= nums[j]:
            tmp[k] = nums[i]
            i += 1
        else:
            tmp[k] = nums[j]
            j += 1
        k += 1
    
    # 处理剩余元素
    while i <= mid:
        tmp[k] = nums[i]
        i += 1
        k += 1
    while j <= right:
        tmp[k] = nums[j]
        j += 1
        k += 1
    
    # 复制回原数组
    for k in range(len(tmp)):
        nums[left + k] = tmp[k]


def merge_sort(nums: list[int], left: int, right: int):
    """归并排序"""
    if left >= right:
        return  # 终止条件：子数组长度为 1
    
    mid = (left + right) // 2  # 计算中点
    merge_sort(nums, left, mid)  # 递归左子数组
    merge_sort(nums, mid + 1, right)  # 递归右子数组
    merge(nums, left, mid, right)  # 合并


# 测试
if __name__ == "__main__":
    nums = [6, 5, 4, 3, 2, 1, 0]
    merge_sort(nums, 0, len(nums) - 1)
    print(f"归并排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n log n)，非自适应排序 |
| 空间复杂度 | O(n)，非原地排序 |
| 稳定性 | 稳定排序 |

## 11.7 堆排序

### 算法思想

基于堆数据结构实现的高效排序算法：

1. 建立大顶堆
2. 将堆顶与堆底交换，已排序元素数量 +1
3. 从堆顶开始执行堆化
4. 循环执行直至排序完成

### Python 代码实现

```python
def swap(nums: list[int], i: int, j: int):
    """交换元素"""
    nums[i], nums[j] = nums[j], nums[i]


def sift_down(nums: list[int], n: int, i: int):
    """从节点 i 开始，从顶到底堆化"""
    while True:
        l = 2 * i + 1  # 左子节点
        r = 2 * i + 2  # 右子节点
        largest = i
        if l < n and nums[l] > nums[largest]:
            largest = l
        if r < n and nums[r] > nums[largest]:
            largest = r
        if largest != i:
            swap(nums, i, largest)
            i = largest
        else:
            break


def heap_sort(nums: list[int]):
    """堆排序"""
    n = len(nums)
    
    # 建堆操作：从最后一个非叶子节点开始下沉
    for i in range(n // 2 - 1, -1, -1):
        sift_down(nums, n, i)
    
    # 循环交换并堆化
    for i in range(n - 1, 0, -1):
        swap(nums, 0, i)  # 将最大元素交换至末尾
        sift_down(nums, i, 0)  # 堆化剩余元素


# 测试
if __name__ == "__main__":
    nums = [4, 1, 3, 1, 5, 2]
    heap_sort(nums)
    print(f"堆排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n log n)，非自适应排序 |
| 空间复杂度 | O(1)，原地排序 |
| 稳定性 | 非稳定排序 |

## 11.8 桶排序

### 算法思想

1. 设置一些具有大小顺序的桶，每个桶对应一个数据范围
2. 将数据平均分配到各个桶中
3. 在每个桶内部分别执行排序
4. 按照桶的顺序合并所有数据

### Python 代码实现

```python
def bucket_sort(nums: list[float]) -> list[float]:
    """桶排序"""
    n = len(nums)
    k = n // 2  # 初始化桶数量
    buckets = [[] for _ in range(k)]
    num_min, num_max = min(nums), max(nums)

    # 将元素分配到各个桶中
    for num in nums:
        idx = int((num - num_min) / (num_max - num_min + 1e-5) * (k - 1))
        buckets[idx].append(num)

    # 对每个桶进行排序
    for bucket in buckets:
        bucket.sort()

    # 合并桶
    res = []
    for bucket in buckets:
        res.extend(bucket)

    return res


# 测试
if __name__ == "__main__":
    nums = [0.4, 0.1, 0.2, 0.3]
    print(f"桶排序完成后 nums = {bucket_sort(nums)}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n + k)，自适应排序 |
| 空间复杂度 | O(n + k)，非原地排序 |
| 稳定性 | 取决于桶内排序算法 |

## 11.9 计数排序

### 算法思想

通过统计元素数量来实现排序，适用于数据范围不大的情况。

### Python 代码实现

```python
def counting_sort(nums: list[int]):
    """计数排序"""
    n = len(nums)
    m = max(nums)
    
    # 创建计数数组
    counter = [0] * (m + 1)
    
    # 统计每个元素的出现次数
    for num in nums:
        counter[num] += 1
    
    # 计算前缀和，确定元素位置
    for i in range(1, m + 1):
        counter[i] += counter[i - 1]
    
    # 倒序遍历，将元素放入正确位置
    res = [0] * n
    for i in range(n - 1, -1, -1):
        num = nums[i]
        idx = counter[num] - 1
        res[idx] = num
        counter[num] -= 1
    
    # 复制回原数组
    for i in range(n):
        nums[i] = res[i]


# 测试
if __name__ == "__main__":
    nums = [1, 0, 1, 2, 0, 4, 0, 2, 2, 4]
    counting_sort(nums)
    print(f"计数排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(n + m)，非自适应排序 |
| 空间复杂度 | O(n + m)，非原地排序 |
| 稳定性 | 稳定排序 |

## 11.10 基数排序

### 算法思想

从最低位到最高位，依次对每一位进行计数排序。高位优先级高于低位，必须先排序低位。

### Python 代码实现

```python
def digit(num: int, exp: int) -> int:
    """获取元素 num 的第 k 位，其中 exp = 10^(k-1)"""
    return (num // exp) % 10


def counting_sort_digit(nums: list[int], exp: int):
    """根据 nums 第 k 位执行计数排序"""
    n = len(nums)
    counter = [0] * 10
    
    # 统计 0~9 各数字的出现次数
    for num in nums:
        d = digit(num, exp)
        counter[d] += 1
    
    # 求前缀和
    for i in range(1, 10):
        counter[i] += counter[i - 1]
    
    # 倒序遍历
    res = [0] * n
    for i in range(n - 1, -1, -1):
        d = digit(nums[i], exp)
        j = counter[d] - 1
        res[j] = nums[i]
        counter[d] -= 1
    
    # 复制回原数组
    for i in range(n):
        nums[i] = res[i]


def radix_sort(nums: list[int]):
    """基数排序"""
    m = max(nums)
    exp = 1
    while exp <= m:
        counting_sort_digit(nums, exp)
        exp *= 10


# 测试
if __name__ == "__main__":
    nums = [123, 456, 789, 321, 654, 987]
    radix_sort(nums)
    print(f"基数排序完成后 nums = {nums}")
```

### 算法特性

| 特性 | 说明 |
|------|------|
| 时间复杂度 | O(nk)，非自适应排序 |
| 空间复杂度 | O(n + d)，非原地排序 |
| 稳定性 | 稳定排序 |

## 排序算法对比总结

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 | 就地性 |
|------|-----------|-----------|--------|--------|
| 选择排序 | O(n²) | O(1) | ❌ | ✅ |
| 冒泡排序 | O(n²) / O(n)* | O(1) | ✅ | ✅ |
| 插入排序 | O(n²) / O(n)* | O(1) | ✅ | ✅ |
| 快速排序 | O(n log n) / O(n²)** | O(log n) | ❌ | ✅ |
| 归并排序 | O(n log n) | O(n) | ✅ | ❌ |
| 堆排序 | O(n log n) | O(1) | ❌ | ✅ |
| 桶排序 | O(n + k) | O(n + k) | ✅/❌ | ❌ |
| 计数排序 | O(n + m) | O(n + m) | ✅ | ❌ |
| 基数排序 | O(nk) | O(n + d) | ✅ | ❌ |

> *: 最佳时间复杂度（已排序数组）
> **: 最差时间复杂度
