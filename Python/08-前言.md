## 0.2 如何使用本书

### 0.2.1 行文风格约定

| 标记 | 含义 |
|------|------|
| 标题后标注 `*` | 选读章节，内容相对困难 |
| **黑体/加粗** | 重点内容和总结性语句 |
| "引号" | 有特指含义的词句 |
| `None` | Python 中表示"空" |

**注释规范**（以 Python 为例）：

```python
"""标题注释，用于标注函数、类、测试样例等"""  # 内容注释，用于详解代码
"""
多行
注释
"""
```

### 0.2.2 动画图解学习法

> 重点和难点知识将主要通过**动画以图解形式展示**，文字则作为解释与补充。

**学习建议**：发现动画图解时，以图为主、以文字为辅，综合两者理解内容。

### 0.2.3 代码实践

> 与阅读代码相比，编写代码的过程往往能带来更多收获。**动手学，才是真的学**。

**运行代码三步曲**：

1. **安装本地编程环境**：参照附录教程安装
2. **克隆代码仓库**：
   ```bash
   git clone https://github.com/krahets/hello-algo.git
   ```
3. **运行源代码**：在 `codes` 文件夹内找到对应文件，一键运行

**额外福利**：网页版支持 Python 代码的**可视化运行**（基于 pythontutor）

### 0.2.4 提问与讨论

- 在评论区提出问题，一般两天内回复
- 阅读评论区可以查漏补缺、激发深入思考
- 帮助他人解答问题，实现共同进步

## 0.3 小结

### 重点回顾

1. ✅ 本书主要受众是算法初学者，已有基础者可用于系统回顾
2. ✅ 内容主要包括复杂度分析、数据结构和算法三部分
3. ✅ 初学阶段阅读一本入门书至关重要，可以少走许多弯路
4. ✅ 动画图解用于介绍重点和难点知识，应给予更多关注
5. ✅ **实践乃学习编程之最佳途径**，强烈建议运行源代码并亲自敲代码
6. ✅ 欢迎在评论区分享疑惑与见解
