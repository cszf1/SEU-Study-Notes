## 5.1 栈

### 5.1.1 栈的定义与特性

**栈（stack）** 是一种遵循**先入后出（LIFO, Last In First Out）** 逻辑的线性数据结构。

可以将栈类比为桌面上的一摞盘子：
- **栈顶**：堆叠元素的顶部
- **栈底**：堆叠元素的底部
- **入栈（push）**：将元素添加到栈顶
- **出栈（pop）**：删除栈顶元素

```
        ┌─────────┐
        │   5     │  ← 栈顶 (top)
        ├─────────┤
        │   3     │
        ├─────────┤
        │   1     │  ← 栈底 (bottom)
        └─────────┘
```

### 5.1.2 栈的常用操作

| 方法 | 描述 | 时间复杂度 |
|------|------|-----------|
| `push()` | 元素入栈（添加至栈顶） | O(1) |
| `pop()` | 栈顶元素出栈 | O(1) |
| `peek()` | 访问栈顶元素 | O(1) |

### 5.1.3 Python 中栈的使用

Python 没有内置的栈类，可以把 `list` 当作栈来使用：

```python
# 初始化栈
stack: list[int] = []

# 元素入栈
stack.append(1)   # [1]
stack.append(3)   # [1, 3]
stack.append(2)   # [1, 3, 2]
stack.append(5)   # [1, 3, 2, 5]
stack.append(4)   # [1, 3, 2, 5, 4]

# 元素出栈
pop: int = stack.pop()  # 4, stack = [1, 3, 2, 5]

# 获取栈的长度
size: int = len(stack)   # 4

# 判断是否为空
is_empty: bool = len(stack) != 0

# 访问栈顶元素
peek: int = stack[-1]    # 5
```

### 5.1.4 栈的实现

#### 1. 基于链表的实现

将链表的头节点视为栈顶，尾节点视为栈底。入栈使用**头插法**，出栈即删除头节点。

```python
from typing import Optional

class ListNode:
    """链表节点"""
    def __init__(self, val: int):
        self.val: int = val
        self.next: Optional['ListNode'] = None

class LinkedListStack:
    """基于链表实现的栈"""
    
    def __init__(self):
        self._peek: Optional[ListNode] = None  # 栈顶节点
        self._size: int = 0                    # 栈的长度
    
    def size(self) -> int:
        return self._size
    
    def is_empty(self) -> bool:
        return self._size == 0
    
    def push(self, val: int):
        """入栈"""
        node = ListNode(val)
        node.next = self._peek  # 新节点指向当前栈顶
        self._peek = node       # 更新栈顶指针
        self._size += 1
    
    def pop(self) -> int:
        """出栈"""
        if self.is_empty():
            raise IndexError("栈为空")
        num = self._peek.val
        self._peek = self._peek.next
        self._size -= 1
        return num
    
    def peek(self) -> int:
        """访问栈顶元素"""
        if self.is_empty():
            raise IndexError("栈为空")
        return self._peek.val
    
    def to_list(self) -> list[int]:
        """转化为列表用于打印"""
        arr = []
        node = self._peek
        while node:
            arr.append(node.val)
            node = node.next
        arr.reverse()
        return arr
```

#### 2. 基于数组的实现

将数组的尾部作为栈顶，入栈和出栈操作都在数组尾部进行。

```python
class ArrayStack:
    """基于数组实现的栈"""
    
    def __init__(self):
        self._stack: list[int] = []
    
    def size(self) -> int:
        return len(self._stack)
    
    def is_empty(self) -> bool:
        return self.size() == 0
    
    def push(self, item: int):
        """入栈"""
        self._stack.append(item)
    
    def pop(self) -> int:
        """出栈"""
        if self.is_empty():
            raise IndexError("栈为空")
        return self._stack.pop()
    
    def peek(self) -> int:
        """访问栈顶元素"""
        if self.is_empty():
            raise IndexError("栈为空")
        return self._stack[-1]
    
    def to_list(self) -> list[int]:
        return self._stack
```

### 5.1.5 两种实现对比

| 维度 | 基于数组 | 基于链表 |
|------|----------|----------|
| **时间效率** | 入栈/出栈 O(1)，但扩容时退化到 O(n) | 稳定 O(1)，但需创建节点 |
| **空间效率** | 可能产生空间浪费 | 节点需存储指针，空间较大 |
| **缓存友好性** | 好（有缓存本地性） | 较差 |

**结论**：
- 触发扩容时，数组实现效率降低，但扩容是低频操作，平均效率更高
- 链表实现提供更稳定的效率表现

### 5.1.6 栈的典型应用

1. **浏览器后退/前进功能**
   - 每打开新网页，入栈
   - 后退操作 = 出栈
   - 需要两个栈配合实现前进和后退

2. **程序内存管理（函数调用栈）**
   - 每次调用函数时，在栈顶添加栈帧
   - 递归函数的递推阶段 = 入栈
   - 递归函数的回溯阶段 = 出栈

3. **撤销/反撤销**
   - 使用两个栈实现

## 5.2 队列

### 5.2.1 队列的定义与特性

**队列（queue）** 是一种遵循**先入先出（FIFO, First In First Out）** 规则的线性数据结构。

模拟排队现象：
- **队首**：队列头部
- **队尾**：队列尾部
- **入队（enqueue）**：将元素加入队尾
- **出队（dequeue）**：删除队首元素

```
入队 →  ┌─────────┬─────────┬─────────┬─────────┐  → 出队
       │   1     │   3     │   2     │   5     │
       └─────────┴─────────┴─────────┴─────────┘
        队首                                    队尾
```

### 5.2.2 队列的常用操作

| 方法名 | 描述 | 时间复杂度 |
|--------|------|-----------|
| `push()` | 元素入队（添加至队尾） | O(1) |
| `pop()` | 队首元素出队 | O(1) |
| `peek()` | 访问队首元素 | O(1) |

### 5.2.3 Python 中队列的使用

Python 推荐使用 `collections.deque`（双向队列）来模拟队列：

```python
from collections import deque

# 初始化队列
que: deque[int] = deque()

# 元素入队
que.append(1)    # [1]
que.append(3)    # [1, 3]
que.append(2)    # [1, 3, 2]
que.append(5)    # [1, 3, 2, 5]
que.append(4)    # [1, 3, 2, 5, 4]

# 访问队首元素
front: int = que[0]    # 1

# 元素出队
pop: int = que.popleft()  # 1, que = [3, 2, 5, 4]

# 获取队列长度
size: int = len(que)     # 4

# 判断队列是否为空
is_empty: bool = len(que) == 0
```

### 5.2.4 队列的实现

#### 1. 基于链表的实现

将链表的头节点视为队首（删除端），尾节点视为队尾（添加端）。

```python
class ListNode:
    def __init__(self, val: int):
        self.val: int = val
        self.next: Optional['ListNode'] = None

class LinkedListQueue:
    """基于链表实现的队列"""
    
    def __init__(self):
        self._front: Optional[ListNode] = None  # 头节点（队首）
        self._rear: Optional[ListNode] = None   # 尾节点（队尾）
        self._size: int = 0
    
    def size(self) -> int:
        return self._size
    
    def is_empty(self) -> bool:
        return self._size == 0
    
    def push(self, num: int):
        """入队"""
        node = ListNode(num)
        if self._rear:
            self._rear.next = node
            self._rear = node
        else:
            self._front = self._rear = node
        self._size += 1
    
    def pop(self) -> int:
        """出队"""
        num = self.peek()
        self._front = self._front.next
        self._size -= 1
        if not self._front:
            self._rear = None
        return num
    
    def peek(self) -> int:
        """访问队首元素"""
        if self.is_empty():
            raise IndexError("队列为空")
        return self._front.val
    
    def to_list(self) -> list[int]:
        """转化为列表"""
        queue = []
        temp = self._front
        while temp:
            queue.append(temp.val)
            temp = temp.next
        return queue
```

#### 2. 基于环形数组的实现

使用 `front` 指针和 `size` 变量，通过取余操作实现环形数组。

```python
class ArrayQueue:
    """基于环形数组实现的队列"""
    
    def __init__(self, size: int):
        self._nums: list[int] = [0] * size   # 存储队列元素的数组
        self._front: int = 0                   # 队首指针
        self._size: int = 0                    # 队列长度
    
    def capacity(self) -> int:
        return len(self._nums)
    
    def size(self) -> int:
        return self._size
    
    def is_empty(self) -> bool:
        return self._size == 0
    
    def push(self, num: int):
        """入队"""
        if self._size == self.capacity():
            raise IndexError("队列已满")
        # 计算队尾索引（环形）
        rear = (self._front + self._size) % self.capacity()
        self._nums[rear] = num
        self._size += 1
    
    def pop(self) -> int:
        """出队"""
        num = self.peek()
        self._front = (self._front + 1) % self.capacity()
        self._size -= 1
        return num
    
    def peek(self) -> int:
        """访问队首元素"""
        if self.is_empty():
            raise IndexError("队列为空")
        return self._nums[self._front]
    
    def to_list(self) -> list[int]:
        res = [0] * self.size()
        j = self._front
        for i in range(self.size()):
            res[i] = self._nums[j % self.capacity()]
            j += 1
        return res
```

### 5.2.5 队列的典型应用

1. **淘宝订单处理**：订单按顺序加入队列，系统按序处理
2. **打印任务队列**：打印机按顺序处理打印任务
3. **餐厅出餐队列**：实现"先来先服务"

## 5.3 双向队列

### 5.3.1 双向队列的定义

**双向队列（deque, double-ended queue）** 提供了更高的灵活性，允许在**头部和尾部**执行元素的添加或删除操作。

```
←── 入队/出队 ──→  ┌─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐  ←── 入队/出队 ──→
                    │   1     │   3     │   5     │   7     │   9     │   2     │
                    └─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘
                     队首                                                     队尾
```

### 5.3.2 双向队列的常用操作

| 方法名 | 描述 | 时间复杂度 |
|--------|------|-----------|
| `push_first()` | 将元素添加至队首 | O(1) |
| `push_last()` | 将元素添加至队尾 | O(1) |
| `pop_first()` | 删除队首元素 | O(1) |
| `pop_last()` | 删除队尾元素 | O(1) |
| `peek_first()` | 访问队首元素 | O(1) |
| `peek_last()` | 访问队尾元素 | O(1) |

### 5.3.3 Python 中双向队列的使用

```python
from collections import deque

# 初始化双向队列
deq: deque[int] = deque()

# 添加元素
deq.append(2)          # 队尾添加：[2]
deq.append(5)          # 队尾添加：[2, 5]
deq.append(4)          # 队尾添加：[2, 5, 4]
deq.appendleft(3)      # 队首添加：[3, 2, 5, 4]
deq.appendleft(1)      # 队首添加：[1, 3, 2, 5, 4]

# 访问元素
front: int = deq[0]    # 队首：1
rear: int = deq[-1]    # 队尾：4

# 删除元素
pop_front: int = deq.popleft()   # 1, deq = [3, 2, 5, 4]
pop_rear: int = deq.pop()        # 4, deq = [3, 2, 5]

# 获取长度和判空
size: int = len(deq)             # 3
is_empty: bool = len(deq) == 0   # False
```

### 5.3.4 双向队列的实现

基于双向链表的实现更为灵活，因为需要同时在两端进行添加和删除操作。

```python
class ListNode:
    def __init__(self, val: int):
        self.val: int = val
        self.next: Optional['ListNode'] = None
        self.prev: Optional['ListNode'] = None

class LinkedListDeque:
    """基于双向链表实现的双向队列"""
    
    def __init__(self):
        self._front: Optional[ListNode] = None  # 头节点
        self._rear: Optional[ListNode] = None    # 尾节点
        self._size: int = 0
    
    def size(self) -> int:
        return self._size
    
    def is_empty(self) -> bool:
        return self._size == 0
    
    def push(self, num: int, is_front: bool):
        """入队"""
        node = ListNode(num)
        if self.is_empty():
            self._front = self._rear = node
        elif is_front:
            # 队首入队
            self._front.prev = node
            node.next = self._front
            self._front = node
        else:
            # 队尾入队
            self._rear.next = node
            node.prev = self._rear
            self._rear = node
        self._size += 1
    
    def push_first(self, num: int):
        self.push(num, True)
    
    def push_last(self, num: int):
        self.push(num, False)
    
    def pop(self, is_front: bool) -> int:
        """出队"""
        if self.is_empty():
            raise IndexError("双向队列为空")
        
        if is_front:
            val = self._front.val
            self._front = self._front.next
            if self._front:
                self._front.prev = None
        else:
            val = self._rear.val
            self._rear = self._rear.prev
            if self._rear:
                self._rear.next = None
        self._size -= 1
        return val
    
    def pop_first(self) -> int:
        return self.pop(True)
    
    def pop_last(self) -> int:
        return self.pop(False)
    
    def peek_first(self) -> int:
        if self.is_empty():
            raise IndexError("双向队列为空")
        return self._front.val
    
    def peek_last(self) -> int:
        if self.is_empty():
            raise IndexError("双向队列为空")
        return self._rear.val
```

### 5.3.5 双向队列的典型应用

双向队列兼具栈与队列的逻辑，可以实现：
- 软件的"撤销"功能（限制步数时，需要在队首删除）
- 需要同时在两端操作的场景

## 5.4 小结

### 重点回顾

1. **栈**：先入后出（LIFO），可通过数组或链表实现
   - 数组实现：效率高，但扩容时退化
   - 链表实现：效率稳定，但需存储指针

2. **队列**：先入先出（FIFO），可通过数组或链表实现
   - 基于环形数组的实现在出队时效率更高

3. **双向队列**：栈 + 队列的组合，两端都可操作
   - 更加灵活，可实现栈和队列的所有应用

### 常见面试问题

| 问题 | 答案 |
|------|------|
| 浏览器前进后退如何实现？ | 两个栈配合使用 |
| 出栈后是否需要释放内存？ | Java/Python 有 GC 自动回收，C/C++ 需要手动释放 |
| 撤销/反撤销如何实现？ | 两个栈：撤销栈 A + 反撤销栈 B |

### 复杂度对比

| 数据结构 | 操作 | 时间复杂度 | 空间复杂度 |
|----------|------|------------|------------|
| **栈** | push/pop/peek | O(1) | O(n) |
| **队列** | push/pop/peek | O(1) | O(n) |
| **双向队列** | 所有操作 | O(1) | O(n) |
