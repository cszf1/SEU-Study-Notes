## 一、多元函数的基本概念

### 1.1 多元函数定义

设 $D$ 是平面上的一个点集，如果对于每个点 $P(x,y) \in D$，变量 $z$ 按照一定法则总有确定的值与之对应，则称 $z$ 是 $x, y$ 的**二元函数**，记作：
$$z = f(x, y)$$

类似地，可以定义三元函数 $u = f(x, y, z)$ 以及 $n$ 元函数。

### 1.2 定义域

函数的定义域是使函数有意义的自变量的取值范围。

**常见约束**：
- 分母不为零
- 根号下非负
- 对数真数大于零
- 反三角函数限制

**例题**：求 $z = \ln(x^2 + y^2 - 1)$ 的定义域

**解**：需要 $x^2 + y^2 - 1 > 0$，即 $x^2 + y^2 > 1$

定义域为：$D = \{(x,y) | x^2 + y^2 > 1\}$（单位圆外部）

## 二、极限与连续

### 2.1 二元函数的极限

**定义**：设函数 $f(x,y)$ 在点 $P_0(x_0, y_0)$ 的某邻域内有定义（$P_0$ 可除外）。如果存在常数 $A$，对于任意 $\varepsilon > 0$，总存在 $\delta > 0$，使得当 $0 < \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta$ 时，有：
$$|f(x,y) - A| < \varepsilon$$

则称 $A$ 为函数 $f(x,y)$ 当 $(x,y) \to (x_0, y_0)$ 时的极限，记作：
$$\lim_{(x,y) \to (x_0,y_0)} f(x,y) = A$$

### 2.2 二重极限与累次极限

**二重极限**：$(x,y)$ 以任意方式趋于 $(x_0, y_0)$

**累次极限**：$\lim_{x \to x_0} \lim_{y \to y_0} f(x,y)$ 或 $\lim_{y \to y_0} \lim_{x \to x_0} f(x,y)$

**重要结论**：
- 若二重极限存在，且累次极限也存在，则两者相等
- 累次极限存在且相等，二重极限不一定存在
- 证明极限不存在：找两条路径极限不同

**例题**：证明 $\lim_{(x,y) \to (0,0)} \frac{xy}{x^2 + y^2}$ 不存在

**解**：
- 沿 $y = 0$ 趋近：$\lim_{x \to 0} \frac{0}{x^2} = 0$
- 沿 $y = x$ 趋近：$\lim_{x \to 0} \frac{x^2}{2x^2} = \frac{1}{2}$

两条路径极限不同，故极限不存在。

### 2.3 连续性

若 $\lim_{(x,y) \to (x_0,y_0)} f(x,y) = f(x_0, y_0)$，则称 $f(x,y)$ 在点 $(x_0, y_0)$ 连续。

**连续函数的性质**：
1. 有界闭区域上的连续函数必有最大值和最小值
2. 有界闭区域上的连续函数必有界
3. 介值定理成立

## 三、偏导数

### 3.1 偏导数的定义

函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 处对 $x$ 的偏导数：
$$\frac{\partial z}{\partial x}\bigg|_{(x_0,y_0)} = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}$$

记作：$f_x(x_0, y_0)$ 或 $z_x(x_0, y_0)$

**几何意义**：曲面 $z = f(x,y)$ 被平面 $y = y_0$ 所截得曲线在点 $M_0$ 处的切线对 $x$ 轴的斜率。

### 3.2 偏导数的计算

**方法**：对哪个变量求偏导，就把其他变量看作常数。

**例题**：设 $z = x^3 y^2 + 2xy$，求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$

**解**：
- $\frac{\partial z}{\partial x} = 3x^2 y^2 + 2y$（把 $y$ 看作常数）
- $\frac{\partial z}{\partial y} = 2x^3 y + 2x$（把 $x$ 看作常数）

### 3.3 高阶偏导数

**二阶偏导数**：
- $\frac{\partial^2 z}{\partial x^2} = \frac{\partial}{\partial x}\left(\frac{\partial z}{\partial x}\right)$ 记作 $z_{xx}$ 或 $f_{xx}$
- $\frac{\partial^2 z}{\partial y^2} = \frac{\partial}{\partial y}\left(\frac{\partial z}{\partial y}\right)$ 记作 $z_{yy}$ 或 $f_{yy}$
- $\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial}{\partial y}\left(\frac{\partial z}{\partial x}\right)$ 记作 $z_{xy}$ 或 $f_{xy}$
- $\frac{\partial^2 z}{\partial y \partial x} = \frac{\partial}{\partial x}\left(\frac{\partial z}{\partial y}\right)$ 记作 $z_{yx}$ 或 $f_{yx}$

**混合偏导数相等的条件**：若 $f_{xy}$ 和 $f_{yx}$ 在区域内连续，则 $f_{xy} = f_{yx}$

## 四、全微分

### 4.1 全微分的定义

若函数 $z = f(x,y)$ 的全增量可表示为：
$$\Delta z = f(x+\Delta x, y+\Delta y) - f(x,y) = A\Delta x + B\Delta y + o(\rho)$$

其中 $\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$，$A, B$ 只与 $x, y$ 有关，则称函数可微，$dz = A\Delta x + B\Delta y$ 为全微分。

### 4.2 可微的条件

**必要条件**：若 $f(x,y)$ 在点 $(x,y)$ 可微，则偏导数存在，且 $A = \frac{\partial z}{\partial x}$，$B = \frac{\partial z}{\partial y}$

**充分条件**：若偏导数 $\frac{\partial z}{\partial x}$，$\frac{\partial z}{\partial y}$ 在点 $(x,y)$ 连续，则函数在该点可微

### 4.3 可微、可导、连续的关系

```
偏导数连续 → 可微 → 偏导数存在
             ↓
           连续
```

**注意**：偏导数存在推不出连续，更推不出可微！

## 五、方向导数与梯度

### 5.1 方向导数

函数 $z = f(x,y)$ 在点 $P(x,y)$ 沿方向 $\vec{l}$（方向角为 $\alpha$）的方向导数：
$$\frac{\partial f}{\partial l} = \frac{\partial f}{\partial x}\cos\alpha + \frac{\partial f}{\partial y}\sin\alpha$$

### 5.2 梯度

$$\nabla f = \text{grad} f = \frac{\partial f}{\partial x}\vec{i} + \frac{\partial f}{\partial y}\vec{j}$$

**重要结论**：
- 梯度方向是函数值增长最快的方向
- 梯度的模是该方向的方向导数
- 方向导数 = 梯度与方向单位向量的点积

$$\frac{\partial f}{\partial l} = \nabla f \cdot \vec{e}_l = |\nabla f| \cos\theta$$

## 六、多元函数的极值

### 6.1 极值的必要条件

若 $z = f(x,y)$ 在点 $(x_0, y_0)$ 有极值且偏导数存在，则：
$$f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0$$

### 6.2 极值的充分条件

设 $f_x(x_0, y_0) = 0$，$f_y(x_0, y_0) = 0$，令：
$$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$
$$\Delta = AC - B^2$$

| 条件 | 结论 |
|------|------|
| $\Delta > 0$ 且 $A < 0$ | 极大值 |
| $\Delta > 0$ 且 $A > 0$ | 极小值 |
| $\Delta < 0$ | 非极值 |
| $\Delta = 0$ | 不确定 |

### 6.3 条件极值与拉格朗日乘数法

求 $z = f(x,y)$ 在条件 $\varphi(x,y) = 0$ 下的极值：

构造拉格朗日函数：
$$L(x,y,\lambda) = f(x,y) + \lambda\varphi(x,y)$$

解方程组：
$$\begin{cases} L_x = f_x + \lambda\varphi_x = 0 \\ L_y = f_y + \lambda\varphi_y = 0 \\ L_\lambda = \varphi(x,y) = 0 \end{cases}$$

## 易错点总结

1. **极限存在性问题**：必须从任意方向趋近，找两条路径极限不同即可证明不存在
2. **偏导数存在 ≠ 连续**：偏导数存在不能推出函数在该点连续
3. **可微性判断**：偏导数存在且连续才能推出可微
4. **混合偏导数**：只有连续时才相等
5. **极值判断**：$\Delta = 0$ 时需用其他方法判断

## 典型例题

**例1**：设 $z = e^{xy}\sin(x+y)$，求 $dz$

**解**：
$$\frac{\partial z}{\partial x} = ye^{xy}\sin(x+y) + e^{xy}\cos(x+y)$$
$$\frac{\partial z}{\partial y} = xe^{xy}\sin(x+y) + e^{xy}\cos(x+y)$$

$$dz = e^{xy}[y\sin(x+y) + \cos(x+y)]dx + e^{xy}[x\sin(x+y) + \cos(x+y)]dy$$

**例2**：求 $z = x^3 + y^3 - 3xy$ 的极值

**解**：
令 $\begin{cases} z_x = 3x^2 - 3y = 0 \\ z_y = 3y^2 - 3x = 0 \end{cases}$

解得驻点：$(0, 0)$ 和 $(1, 1)$

$A = 6x$，$B = -3$，$C = 6y$

在 $(0,0)$：$\Delta = -9 < 0$，非极值

在 $(1,1)$：$A = 6 > 0$，$\Delta = 36 - 9 = 27 > 0$，极小值，$z_{min} = -1$
