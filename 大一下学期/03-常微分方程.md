## 一、微分方程的基本概念

### 1.1 微分方程的定义

**微分方程**是含有未知函数及其导数的方程。

**阶数**：方程中未知函数的最高阶导数的阶数。

- **一阶微分方程**：$y' = f(x, y)$
- **二阶微分方程**：$y'' = f(x, y, y')$

### 1.2 微分方程的解

**通解**：含有任意常数，且任意常数的个数等于方程阶数的解。

$$y = f(x, C_1, C_2, \cdots, C_n)$$

**特解**：确定了任意常数之后的解。

**初始条件**：用于确定通解中任意常数的条件。

$$y(x_0) = y_0, \quad y'(x_0) = y_0', \cdots$$

### 1.3 例题

**例1**：验证 $y = Ce^x$ 是微分方程 $y' = y$ 的通解。

**解**：
$$y' = Ce^x = y$$

代入方程成立，且含有一个任意常数，故为通解。

**例2**：求 $y' = 2x$ 满足 $y(0) = 1$ 的特解。

**解**：
通解：$y = x^2 + C$

代入初始条件 $y(0) = 1$ 得 $C = 1$

特解：$y = x^2 + 1$

## 二、一阶微分方程

### 2.1 可分离变量的微分方程

**形式**：$\frac{dy}{dx} = f(x) \cdot g(y)$

**解法**：分离变量，两边积分

$$\int \frac{dy}{g(y)} = \int f(x) \, dx$$

**例题**：求 $\frac{dy}{dx} = 2xy$ 的通解。

**解**：
$$\frac{dy}{y} = 2x \, dx$$

两边积分：

$$\ln|y| = x^2 + C_1$$

$$y = \pm e^{x^2 + C_1} = Ce^{x^2}$$

（其中 $C = \pm e^{C_1}$ 为任意非零常数，令 $C = 0$ 可得平凡解 $y = 0$）

### 2.2 齐次方程

**形式**：$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$

**解法**：令 $u = \frac{y}{x}$，则 $y = ux$，$y' = u + xu'$

代入方程化为可分离变量方程。

**例题**：求 $\frac{dy}{dx} = \frac{y}{x} + \tan\left(\frac{y}{x}\right)$ 的通解。

**解**：

令 $u = \frac{y}{x}$，则 $y' = u + xu'$

$$u + xu' = u + \tan u$$

$$xu' = \tan u$$

$$\frac{du}{\tan u} = \frac{dx}{x}$$

$$\ln|\sin u| = \ln|x| + C_1$$

$$\sin u = Cx$$

$$\sin\frac{y}{x} = Cx$$

### 2.3 一阶线性微分方程

**标准形式**：$y' + P(x)y = Q(x)$

**解法**：使用积分因子法

积分因子：$\mu(x) = e^{\int P(x) \, dx}$

通解公式：

$$y = e^{-\int P(x) \, dx} \left( \int Q(x) e^{\int P(x) \, dx} \, dx + C \right)$$

**例题**：求 $y' + \frac{1}{x}y = x$ 的通解。

**解**：

$P(x) = \frac{1}{x}$，$Q(x) = x$

积分因子：$\mu = e^{\int \frac{1}{x} dx} = e^{\ln|x|} = x$

两边乘以 $x$：

$$xy' + y = x^2$$

$$(xy)' = x^2$$

积分：

$$xy = \int x^2 \, dx = \frac{x^3}{3} + C$$

$$y = \frac{x^2}{3} + \frac{C}{x}$$

### 2.4 伯努利方程

**形式**：$y' + P(x)y = Q(x)y^n$（$n \neq 0, 1$）

**解法**：令 $z = y^{1-n}$，化为线性方程

**例题**：求 $y' + \frac{1}{x}y = xy^2$ 的通解。

**解**：

令 $z = y^{1-2} = y^{-1}$，则 $z' = -y^{-2}y'$

方程两边乘以 $-y^{-2}$：

$$-\frac{y'}{y^2} - \frac{1}{x}\frac{1}{y} = -x$$

$$z' - \frac{1}{x}z = -x$$

$P(x) = -\frac{1}{x}$，$Q(x) = -x$

积分因子：$\mu = e^{\int -\frac{1}{x} dx} = e^{-\ln|x|} = \frac{1}{x}$

$$\frac{z'}{x} - \frac{z}{x^2} = -1$$

$$\left(\frac{z}{x}\right)' = -1$$

$$\frac{z}{x} = -x + C$$

$$z = -x^2 + Cx$$

$$y^{-1} = -x^2 + Cx$$

$$y = \frac{1}{Cx - x^2}$$

## 三、可降阶的高阶微分方程

### 3.1 $y^{(n)} = f(x)$ 型

**解法**：连续积分 $n$ 次

**例题**：求 $y''' = x$ 的通解。

**解**：

$$y'' = \int x \, dx = \frac{x^2}{2} + C_1$$

$$y' = \int \left(\frac{x^2}{2} + C_1\right) dx = \frac{x^3}{6} + C_1 x + C_2$$

$$y = \int \left(\frac{x^3}{6} + C_1 x + C_2\right) dx = \frac{x^4}{24} + \frac{C_1}{2}x^2 + C_2 x + C_3$$

### 3.2 $y'' = f(x, y')$ 型（缺 $y$）

**解法**：令 $y' = p$，则 $y'' = p'$

方程化为 $p' = f(x, p)$，按一阶方程求解。

**例题**：求 $y'' = 2x y'$ 满足 $y(0) = 1, y'(0) = 0$ 的特解。

**解**：

令 $p = y'$，则 $p' = 2xp$

$$\frac{dp}{p} = 2x \, dx$$

$$\ln|p| = x^2 + C_1$$

由 $y'(0) = 0$ 得 $C_1 = 0$，故 $p = 0$ 或 $p = e^{x^2} - 1$ 等。

实际上：$p = Ce^{x^2}$，由 $p(0) = 0$ 得 $C = 0$，所以 $y' = 0$

积分得 $y = C_2 = 1$

### 3.3 $y'' = f(y, y')$ 型（缺 $x$）

**解法**：令 $y' = p$，则 $y'' = p\frac{dp}{dy}$

方程化为 $p\frac{dp}{dy} = f(y, p)$，按一阶方程求解。

**例题**：求 $yy'' = (y')^2$ 的通解。

**解**：

令 $p = y'$，则 $y'' = p\frac{dp}{dy}$

$$yp\frac{dp}{dy} = p^2$$

若 $p \neq 0$：

$$\frac{dp}{p} = \frac{dy}{y}$$

$$\ln|p| = \ln|y| + C_1$$

$$p = C_2 y$$

即 $\frac{dy}{dx} = C_2 y$

$$\frac{dy}{y} = C_2 dx$$

$$\ln|y| = C_2 x + C_3$$

$$y = C_4 e^{C_2 x}$$

（$C_4 = e^{C_3}$）

## 四、二阶线性微分方程

### 4.1 二阶线性微分方程的一般形式

$$y'' + P(x)y' + Q(x)y = f(x)$$

- 若 $f(x) = 0$，称为**齐次方程**
- 若 $f(x) \neq 0$，称为**非齐次方程**

### 4.2 齐次方程解的结构

**定理**：若 $y_1, y_2$ 是齐次方程的两个线性无关的特解，则通解为：

$$y = C_1 y_1 + C_2 y_2$$

### 4.3 非齐次方程解的结构

**定理**：若 $y^*$ 是非齐次方程的一个特解，$\tilde{y}$ 是对应齐次方程的通解，则非齐次方程的通解为：

$$y = \tilde{y} + y^*$$

**叠加原理**：若 $f(x) = f_1(x) + f_2(x)$，则 $y^* = y_1^* + y_2^*$（$y_i^*$ 为对应 $f_i$ 的特解）

## 五、常系数齐次线性微分方程

### 5.1 特征方程

对于 $y'' + py' + qy = 0$，设 $y = e^{rx}$，代入得特征方程：

$$r^2 + pr + q = 0$$

### 5.2 通解的三种情况

| 特征根情况 | 通解形式 |
|:---|:---|
| 两个不相等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| 两个相等实根 $r_1 = r_2 = r$ | $y = (C_1 + C_2 x)e^{rx}$ |
| 一对共轭复根 $r = \alpha \pm \beta i$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

### 5.3 例题

**例1**：求 $y'' - 3y' + 2y = 0$ 的通解。

**解**：

特征方程：$r^2 - 3r + 2 = 0$

$(r-1)(r-2) = 0$

$r_1 = 1, r_2 = 2$

通解：$y = C_1 e^x + C_2 e^{2x}$

**例2**：求 $y'' - 4y' + 4y = 0$ 的通解。

**解**：

特征方程：$r^2 - 4r + 4 = 0$

$(r-2)^2 = 0$

$r_1 = r_2 = 2$

通解：$y = (C_1 + C_2 x)e^{2x}$

**例3**：求 $y'' + 2y' + 5y = 0$ 的通解。

**解**：

特征方程：$r^2 + 2r + 5 = 0$

$r = \frac{-2 \pm \sqrt{4-20}}{2} = -1 \pm 2i$

通解：$y = e^{-x}(C_1 \cos 2x + C_2 \sin 2x)$

## 六、常系数非齐次线性微分方程

### 6.1 待定系数法

根据 $f(x)$ 的形式设出特解形式，代入方程确定系数。

#### 情况一：$f(x) = P_n(x)e^{\alpha x}$

**特解形式**：$y^* = x^k Q_n(x)e^{\alpha x}$

其中 $Q_n(x)$ 与 $P_n(x)$ 同次多项式，$k$ 为 $\alpha$ 与特征根的重数。

| $\alpha$ 与特征根关系 | $k$ 值 |
|:---|:---|
| 不是特征根 | $k = 0$ |
| 是单特征根 | $k = 1$ |
| 是二重特征根 | $k = 2$ |

#### 情况二：$f(x) = e^{\alpha x}[P_m(x)\cos\beta x + P_n(x)\sin\beta x]$

**特解形式**：$y^* = x^k e^{\alpha x}[R_l(x)\cos\beta x + S_l(x)\sin\beta x]$

其中 $l = \max(m, n)$，$k$ 为 $\alpha + i\beta$ 与特征根的重数。

### 6.2 例题

**例1**：求 $y'' - 2y' - 3y = 3xe^{4x}$ 的通解。

**解**：

1. 对应齐次方程：$y'' - 2y' - 3y = 0$

   特征方程：$r^2 - 2r - 3 = 0$

   $r_1 = 3, r_2 = -1$

   齐次通解：$\tilde{y} = C_1 e^{3x} + C_2 e^{-x}$

2. 求特解

   $f(x) = 3xe^{4x}$，$P_1(x) = 3x$，$\alpha = 4$

   4 不是特征根，$k = 0$

   设 $y^* = (Ax + B)e^{4x}$

   $$(y^*)' = Ae^{4x} + 4(Ax+B)e^{4x} = (4Ax + A + 4B)e^{4x}$$

   $$(y^*)'' = 4Ae^{4x} + 4(4Ax + A + 4B)e^{4x} = (16Ax + 8A + 16B)e^{4x}$$

   代入：

   $(16Ax + 8A + 16B) - 2(4Ax + A + 4B) - 3(Ax + B) = 3x$

   $16Ax + 8A + 16B - 8Ax - 2A - 8B - 3Ax - 3B = 3x$

   $5Ax + (6A + 5B) = 3x$

   $$\begin{cases} 5A = 3 \\ 6A + 5B = 0 \end{cases} \Rightarrow \begin{cases} A = \frac{3}{5} \\ B = -\frac{18}{25} \end{cases}$$

   $$y^* = \left(\frac{3}{5}x - \frac{18}{25}\right)e^{4x}$$

3. 通解：$y = C_1 e^{3x} + C_2 e^{-x} + \left(\frac{3}{5}x - \frac{18}{25}\right)e^{4x}$

## 七、欧拉方程

### 7.1 定义

形如 $x^n y^{(n)} + a_1 x^{n-1} y^{(n-1)} + \cdots + a_{n-1} xy' + a_n y = 0$ 的方程称为欧拉方程。

### 7.2 解法

令 $x = e^t$（或 $t = \ln x$），则：

$$xy' = \frac{dy}{dt}, \quad x^2 y'' = \frac{d^2 y}{dt^2} - \frac{dy}{dt}$$

转化为常系数线性微分方程。

### 7.3 例题

**例**：求 $x^2 y'' + 3xy' + y = 0$ 的通解。

**解**：

令 $x = e^t$，则 $x^2 y'' = \frac{d^2 y}{dt^2} - \frac{dy}{dt}$，$xy' = \frac{dy}{dt}$

方程化为：

$$\frac{d^2 y}{dt^2} - \frac{dy}{dt} + 3\frac{dy}{dt} + y = 0$$

$$\frac{d^2 y}{dt^2} + 2\frac{dy}{dt} + y = 0$$

特征方程：$r^2 + 2r + 1 = 0$，$(r+1)^2 = 0$，$r_1 = r_2 = -1$

$$y = (C_1 + C_2 t)e^{-t}$$

回代 $t = \ln x$：

$$y = \left(C_1 + C_2 \ln x\right)\frac{1}{x}$$

## 八、微分方程的应用

### 8.1 建立微分方程的一般步骤

1. 确定研究对象，建立坐标系
2. 分析微小段时间内的变化
3. 利用物理/几何规律列出等式
4. 写出微分方程及初始条件

### 8.2 典型应用

**例**：设降落伞从飞机上降落，已知空气阻力与速度成正比，比例系数为 $k$（$k > 0$），降落伞质量为 $m$，初始速度为 0。求速度随时间的变化规律。

**解**：

受力分析：重力 $mg$ 向下，空气阻力 $kv$ 向上

$$m\frac{dv}{dt} = mg - kv$$

$$\frac{dv}{dt} = g - \frac{k}{m}v$$

$$\frac{dv}{dt} + \frac{k}{m}v = g$$

$$P(t) = \frac{k}{m}, \quad Q(t) = g$$

积分因子：$\mu = e^{\int \frac{k}{m} dt} = e^{\frac{k}{m}t}$

$$\frac{d}{dt}(v \cdot e^{\frac{k}{m}t}) = ge^{\frac{k}{m}t}$$

积分：

$$v \cdot e^{\frac{k}{m}t} = \frac{mg}{k}e^{\frac{k}{m}t} + C$$

$$v = \frac{mg}{k} + Ce^{-\frac{k}{m}t}$$

初始条件 $v(0) = 0$：

$$0 = \frac{mg}{k} + C \Rightarrow C = -\frac{mg}{k}$$

$$v(t) = \frac{mg}{k}\left(1 - e^{-\frac{k}{m}t}\right)$$

**物理意义**：当 $t \to \infty$ 时，$v \to \frac{mg}{k}$，即趋于极限速度。

## 九、易错点总结

1. **通解中的任意常数**：一阶方程有 1 个任意常数，二阶方程有 2 个

2. **分离变量积分**：注意 $|y|$ 的处理，最后可去掉绝对值并允许常数 $C$ 取任意实数

3. **线性方程积分因子**：$\mu = e^{\int P(x)dx}$，不要忘记乘回去

4. **伯努利方程**：一定要做变量代换 $z = y^{1-n}$，不要直接套线性公式

5. **特征方程**：$e^{rx}$ 代入后，注意 $r$ 前面的系数是 +1

6. **特解形式选择**：仔细判断 $\alpha$ 或 $\alpha + i\beta$ 是否为特征根

7. **欧拉方程**：回代时别忘了 $t = \ln x$

## 典型例题

**综合题**：求 $y'' - 5y' + 6y = xe^{2x}$ 的通解。

**解**：

1. 齐次方程：$y'' - 5y' + 6y = 0$

   特征方程：$r^2 - 5r + 6 = 0$

   $(r-2)(r-3) = 0$，$r_1 = 2, r_2 = 3$

   齐次通解：$\tilde{y} = C_1 e^{2x} + C_2 e^{3x}$

2. 求特解

   $f(x) = xe^{2x}$，$\alpha = 2$ 是单特征根

   设 $y^* = x \cdot (Ax + B)e^{2x} = (Ax^2 + Bx)e^{2x}$

   计算：

   $$y^* = (Ax^2 + Bx)e^{2x}$$

   $$(y^*)' = (2Ax + B)e^{2x} + 2(Ax^2 + Bx)e^{2x} = (2Ax^2 + (2B+2A)x + B)e^{2x}$$

   $$(y^*)'' = [4Ax + (2B+2A) + 4Ax^2 + 4(2Bx+2A)x + 4Bx + 2B]e^{2x}$$

   实际上直接代入计算更清晰。设 $y^* = (Ax^2 + Bx)e^{2x}$：

   $y^*' = (2Ax + B + 2Ax^2 + 2Bx)e^{2x} = (2Ax^2 + (2A+2B)x + B)e^{2x}$

   $y^*'' = [4Ax + 2A + 2B + 4Ax + 4(2A+2B)x + 4Bx + 2B]e^{2x}$...

   
   代入原方程整理后得到：$-2Ax + (2A - B) = x$
   
   比较系数：$-2A = 1$ 且 $2A - B = 0$，解得 $A = -\frac{1}{2}$，$B = -1$
   
   因此特解为 $y^* = -\left(\frac{x^2}{2} + x\right)e^{2x}$

3. 最终通解为 $y = C_1 e^{2x} + C_2 e^{3x} - \left(\frac{x^2}{2} + x\right)e^{2x}$
